## オブジェクト配列

### 宣言方法

**クラス名 オブジェクト名[サイズ];**

例）Circle pizza[10];

<br/>

### 使用目的

- 複数のオブジェクトを1つの名前で管理

<br/>

### コンストラクタとデストラクタ

- 各引数ごとにコンストラクタとデストラクタが呼び出される
- デフォルトコンストラクタのみが呼び出される
- デストラクタは、生成された順序と逆の順序で呼び出される
- 呼び出し順序のサンプルコード）

  ~~~c++
  class ArrayCircleClass{
  private:
 int radius;
  public:
 ArrayCircleClass() { radius = 1; }
 ArrayCircleClass(int r) { radius = r; }
 
 float GetArea();
      
      ~ArrayCircleClass() { cout << radius << " サイズの円が削除されました。" << endl; }
  };
 
 float ArrayCircleClass::GetArea() { return 3.14 * radius * radius; }
  
  int main()
  {
 ArrayCircleClass arrayCircle[3] = { ArrayCircleClass(), ArrayCircleClass(5), ArrayCircleClass(10)};
 
 for(int i=0; i< 3; i++)
 cout << " " << arrayCircle[i].GetArea() << endl;
  }
  ~~~

  **結果**

  ~~~
   3.14
   78.5
   314
  半径10の円が削除されました。
  半径5の円が削除されました。
  半径1の円が削除されました。
  ~~~

  サンプルコードでは、コンストラクタはarrayCircle[0]、arrayCircle[1]、arrayCircle[2]の順に呼び出されます

  消滅子は生成子とは逆に、arrayCircle[2]、arrayCircle[1]、arrayCircle[0]の順に呼び出される

<br/>

### 初期化

- オブジェクト配列のコンストラクタの指定と初期化

例）

~~~c++
Circle circleArray[3] = [Circle(), Circle(10), Circle(20)];
~~~

この例では、circleArray[0]であるCircle()はデフォルトコンストラクタを呼び出す。

circleArray[1]、circleArray[2]はパラメータ渡しコンストラクタを呼び出し、それぞれCircle(10)、Circle(20)を呼び出します

## 動的メモリ

### 動的メモリと静的メモリの違い

#### 動的メモリ：

- プログラム作成時には、固定サイズのメモリを使用する

#### 静的メモリ：

- 実行中に必要なメモリを割り当てる（ヒープメモリ）

### 動的メモリの使用方法

- 割り当て: new

- 戻り値: delete

  <br/>

- **形式**

  - **データ型 *ポインタ変数 = new データ型;**
    - 配列の場合は **new データ型[サイズ];** と記述する必要がある
  - **delete ポインタ変数;**
    - 配列の場合は **delete[] ポインタ変数名;** と記述する必要がある

- 例）

~~~c++
int pIntVar = new int; // int型のメモリを動的に割り当て
Circle *pCircleVar = new Circle; // Circleクラスのメモリを動的に割り当て

delete pIntVar; //割り当てられた整数領域を解放
delete pCircleVar; //割り当てられたオブジェクト領域を解放
~~~

## 自己参照ポインタ

### これ

- 自身を指すポインタであり、メンバー関数内でのみ使用される

  - 例）

    ~~~c++
    Circle::Circle(int radius) { this->radius = radius; }
    ~~~

- 他のオブジェクトのthisとは共有されない

- パラメータとメンバー変数の名前が同じ場合に使用される

- 自身のオブジェクトのアドレスを返すために使用

- staticメンバー関数では使用できない

<br/>

#### コンパイラによるコード変換

**コンパイル前**

~~~c++
class Circle{
    int radius;
public:
    Circle(int radius) { 
 this->radius = radius;
 }
};
~~~

**コンパイル後**

~~~c++
class Circle{
    int radius;
public:
    Circle(Circle* this, int radius){
    this->radius = radius;
    }
};
~~~
