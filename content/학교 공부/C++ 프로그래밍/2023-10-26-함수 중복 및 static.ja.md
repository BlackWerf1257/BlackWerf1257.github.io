## 参照

### 参照による呼び出し

#### 関数の引数を参照型として宣言する

##### 形式

- **変数型 関数名(変数型 &変数名)**

- 例

~~~c++
void testFunc(int &n1, int &n2);
~~~

#### 参照の呼び出しが必要な場合

- 返されるデータが複数種類ある場合
  - 値またはエラー情報

<br/>

#### 参照を使用した返却

- 例

  ~~~c++
  #include <iostream>
  using namespace std;
 
 bool average(int arr[], int size, int& avg)
  {
 if (size <= 0) return false;
 int sum = 0;
  
      for (int i = 0; i < size; i++)
 sum += arr[i];
 
 avg = sum / size;
 return true;
  }
 
 int main() {
  start:
 int arrNum, avg;
 cout << "数字の個数を入力してください>>";
      cin >> arrNum;
 int* numArr = new int[arrNum];
 
 for (int i = 0; i < arrNum; i++)
 {
 cout << i << "番目の数字を入力してください>>";
          cin >> numArr[i];
 }
 
 if (average(numArr, arrNum, avg)) cout << avg << endl;
 else {
 cout << "数字の数が0以下です。最初に戻ります" << endl;
 goto start;
 }
  }
  ~~~

  **結果**

  ~~~
  数字の個数を入力してください>>3
  1つ目の数字を入力してください>>6
  2つ目の数字を入力してください>>2
  3つ目の数字を入力してください>>1
  3
  ~~~

#### 参照を用いたオブジェクトの受け渡し

- 例

  ~~~c++
  #include <iostream>
  using namespace std;
 
 class Circle{
 private:
 int rad;
 public:
 Circle(): Circle(5){}
 Circle(int r) { rad = r; 
              cout << "クラスが生成されました。\n半径 : " << rad << endl;
 }
 ~Circle(){
 cout << "デストラクタが呼び出されました。\n半径 : " << rad << endl; }
 int GetRad(){ return rad; }
 int SetRad(int r){return rad = r; } 
  };
  void IncreaseRad(Circle &c) {
 int rad = c.GetRad();
 c.SetRad(++rad);
  }
  int main() {
 Circle circle;
 IncreaseRad(circle);
 cout << circle.GetRad() << endl;
  }
  ~~~

  **結果**

  ~~~ 
  クラスが生成されました。
  半径：5
  6
  デストラクタが呼び出されました。
  半径：6
  ~~~

  ### 参照戻り値

  #### C言語の関数の戻り値

  - 関数は必ず値のみを返す

  -  基本型: int、char、double…

  -  ポインタ値

   <br/>

  #### C++の関数の戻り値

  - 関数は値だけでなく、参照も返すことができる

  - 参照の返却
    - 変数など、既存の領域への参照を返す
 - 変数の値を返すことではない

<br>

### オブジェクトのコピー

 #### 浅いコピー（Shallow Copy）

- ポインタ**のみ**をコピー
- 動的に割り当てられた変数のアドレス値を共有する

#### ディープコピー（Deep Copy）

- ポインタのメモリ**も**コピーされる
- コピー後は個別のオブジェクトとなるため、コピー対象が変わっても影響はない

 <br/>

 ### コピーコンストラクタ

#### 特徴

- オブジェクトのコピー作成時に呼び出されるコンストラクタ

- 1つのクラスにつき**1つ**のみ宣言可能（関数の重複は不可）

- クラスへの参照パラメータを持つ

#### 実装方法

~~~c++
class クラス名{
  クラス名(const クラス名& オブジェクト名); //コピーコンストラクタの宣言
}

//コピーコンストラクタの実装
クラス名::クラス名(const クラス名& オブジェクト名);
~~~

**例**

~~~c++
#include <iostream>
using namespace std;

class Square {
private:
    int length;
public:
  Square(const Square& s);
    Square() :Square(5) {};
    Square(int l) { length = l; }
    int GetArea() { return length * length; }
};

//コピーコンストラクタ
Square::Square(const Square& s) {
    this->length = s.length;
    cout << "コピーコンストラクタが実行されました" << endl;
}

int main() {
    Square b1;
    Square b2(b1);

    cout << "b1の面積>>" << b1.GetArea() << endl;
    cout << "b2の面積>>" << b2.GetArea() << endl;
}
~~~

**結果**

~~~
コピーコンストラクタが実行されました
b1の面積>>25
b2の面積>>25
~~~

### 多型（Polymorphism）

#### 関数の重複

- 同じ名前の関数を使用可能

- **ただし**、変数の型または個数に違いがある必要がある

- 戻り値の型は関数の重複とは関係がない

- 同じ名前の関数を使用 -> 関数名を区別する必要がないという利点

 

#### オーバーロード

- 関数間の重複

- 例

  ~~~C++
  //正しい例
  int(int a, int b) { return a+b;}
  int(char a, char b) {return a+b;}
  
  //誤った例
  double(int a, int b) { return a+b;} //戻り値の型は関数のオーバーロードとは関係ありません
  int(char a, char b) {return a+b;}
  ~~~

  

#### オーバーライド

- 継承されたオブジェクト間の関数の重複

- 消滅子は関数と重複できない

- パラメータなし

<br/>

#### デフォルトパラメータ

##### 特徴

-  引数の値を指定しなかった場合に、呼び出された関数で設定されるデフォルト値

##### 例

 ~~~c++
 //宣言部
 void TestVoid(int id=0, string name="Test"){
 cout << "ID>>" << id << endl;
 cout << "名前>>" << name << endl;
 }
 
 //呼び出し部
 TestVoid();
 TestVoid(2021, "キム");


//結果
 ID>>0
 名前>>Test
 ID>>2021
 名前>>キム
 ~~~

##### 注意事項

-  一般パラメータの前に宣言することはできません

- 一般パラメータの後に宣言する必要がある
