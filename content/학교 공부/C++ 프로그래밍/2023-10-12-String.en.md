### Object Substitution and Function Return

#### Object Substitution

- **Interchangeable between objects**

- Example)

  ~~~c++
  class Circle {
  private:
 int radius;
  public:
 Circle() {
 radius = 1;
 }
 Circle(int r) {
 radius = r;
 }
 int getRadius() {
 return radius;
 }
  };
  
  
  int main()
  {
 Circle c1(5);
 Circle c2(30);
 cout << "c1 radius before substitution: " << c1.getRadius() << endl;
 cout << "c2 radius before substitution: " << c2.getRadius() << endl;
      c1 = c2;
 cout << "c1 radius after swap: " << c1.getRadius() << endl;
 cout << "c2 radius after swap: " << c2.getRadius() << endl;
  }
  ~~~

  **Results**

  ~~~
  c1 radius before replacement: 5
  c2 radius before replacement: 30
  c1 radius after replacement: 30
  c2 radius after replacement: 30
  ~~~

<br/>

- In the example code, c1 has a radius value of 5, and c2 has a radius value of 30.

  Before the substitution, the radii of c1 and c2 are generated individually and have initial values of 5 and 30, respectively.

  After substituting the value of c2 into c1 using c1=c2, c1 takes on the value of 30, just like c2.

<br/>

#### Returning Objects

- **Objects Returning Functions**
- Example

~~~c++
class Circle {
private:
    int radius;
public:
    Circle() {
 radius = 1;
    }
    Circle(int r) {
 radius = r;
    }
    int getRadius() {
 return radius;
    }
};

Circle getCircle() {
    Circle tmpCircle(30);
    return tmpCircle;
}

int main()
{
    Circle c1(5);
    cout << "Radius of c1 before the object is returned >>" << c1.getRadius() << endl;
    c1 = getCircle();
    cout << "Radius of c1 after the object is returned >>" << c1.getRadius() << endl;
}
~~~

**Results**

~~~
Before returning the object: c1's radius >>5
After returning the object: c1's radius >>30
~~~

<br/>

- In the sample code, there is a c1 object with a value of 5 and a getCircle object.

The `getCircle` object returns a `tmpCircle` object with a radius of 30.

When `c1 = getCircle()` is used to assign the function of the `getCircle` object to `c1`, `c1` takes on the value of 30, which is the radius of `tmpCircle` within the `getCircle` object.

## References

### Variable References

- Syntax: **&variable_name = reference_variable_name;**

- Example)

  ~~~C++
  int main(){
 
 int a = 20;
  int &refv = a;
 
 cout << "a >>" << a << endl; 
 cout << "refv >>" << refv << endl;
  }
  ~~~

  **Results**

  ~~~
  a >>20
  refv >>20
  ~~~

  <br/>

  - The variable a is declared and initialized to 20.

    refv is created to reference the variable a and is initialized to 20.

  <br/>

### Object References

~~~c++
class Circle {
private:
    int radius;
public:
    Circle() {
 radius = 1;
    }
    Circle(int r) {
 radius = r;
    }
    int getRadius() {
 return radius;
    }
    int setRadius(int radius) {
 return this->radius = radius;
    }
};


int main()
{
    Circle circle;
    Circle& refCircle = circle;
    refCircle.setRadius(20);
    cout << "circle의 radius: " << circle.getRadius() << endl;
    cout << "refCircle radius: " << refCircle.getRadius() << endl;
}
~~~

**Results**

~~~
circle의 radius: 20
refCircle radius: 20
~~~

<br/>

- refCircle references circle, which is also a Circle object.

  The `circle` is instantiated without parameters, so the value of `radius` is 1.

  When `setRadius(20)` is executed on `refCircle`, the value of `circle` is also updated, and the radius of both `refCircle` and `circle` becomes 20.
