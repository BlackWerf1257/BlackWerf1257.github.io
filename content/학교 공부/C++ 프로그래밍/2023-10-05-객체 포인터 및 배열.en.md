## Object Arrays

### How to Declare

**ClassName ObjectName[size];**

예시) Circle pizza[10];

<br/>

### Purpose

- Manage multiple objects under a single name

<br/>

### Constructors and Destructors

- The constructor and destructor are called for each parameter
- Only the default constructor is called
- Destructors are called in the reverse order of creation
- Example code showing the call order)

  ~~~c++
  class ArrayCircleClass{
  private:
 int radius;
  public:
 ArrayCircleClass() { radius = 1; }
 ArrayCircleClass(int r) { radius = r; }
 
 float GetArea();
      
      ~ArrayCircleClass() { cout << radius << " 크기의 원 제거됨." << endl; }
  };
 
 float ArrayCircleClass::GetArea() { return 3.14 * radius * radius; }
  
  int main()
  {
 ArrayCircleClass arrayCircle[3] = { ArrayCircleClass(), ArrayCircleClass(5), ArrayCircleClass(10)};
 
 for(int i=0; i< 3; i++)
 cout << " " << arrayCircle[i].GetArea() << endl;
  }
  ~~~

  **Results**

  ~~~
   3.14
   78.5
   314
  Circle with a radius of 10 removed.
  Circle with a radius of 5 removed.
  Circle with a radius of 1 removed.
  ~~~

  In the sample code, the constructors are called in the order arrayCircle[0], arrayCircle[1], and arrayCircle[2].

  Unlike the creator, the destructor is called in the order arrayCircle[2], arrayCircle[1], and arrayCircle[0].

<br/>

### Initialization

- Specifying constructors and initializing object arrays

Example)

~~~c++
Circle circleArray[3] = [Circle(), Circle(10), Circle(20)];
~~~

In this example, `Circle()` in `circleArray[0]` calls the default constructor.

circleArray[1] and circleArray[2] call the constructor with parameters, invoking Circle(10) and Circle(20), respectively

## Dynamic Memory

### The Difference Between Dynamic and Static Memory

#### Dynamic Memory:

- When writing programs, fixed-size memory is used

#### Static memory:

- Allocates the memory required during execution (heap memory)

### How to Use Dynamic Memory

- Allocation: new

- Return type: delete

  <br/>

- **Format**

  - **DataType *pointerVariable = new DataType;**
    - For arrays, use **new DataType[size];**
  - **delete pointerVariable;**
    - For arrays, use **delete[] pointerVariableName;**

- Example)

~~~c++
int pIntVar = new int; // Dynamically allocate memory of type int
Circle *pCircleVar = new Circle; // Dynamically allocate memory of type Circle

delete pIntVar; // Free the allocated integer memory
delete pCircleVar; // Free the allocated object memory
~~~

## Self-referencing pointers

### this

- Used only within member functions as a pointer to itself

  - Example)

    ~~~c++
    Circle::Circle(int radius) { this->radius = radius; }
    ~~~

- Not shared with the `this` of other objects

- Used when a parameter and a member variable have the same name

- Used to return the address of the object itself

- Cannot be used in static member functions

<br/>

#### Code Transformation by the Compiler

**Before compilation**

~~~c++
class Circle{
    int radius;
public:
    Circle(int radius) { 
 this->radius = radius;
 }
};
~~~

**After compilation**

~~~c++
class Circle{
    int radius;
public:
    Circle(Circle* this, int radius){
 this->radius = radius;
    }
};
~~~
