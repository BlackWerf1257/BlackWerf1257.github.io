## 文字列

### 特徴

- C++の標準ライブラリであり、可変長文字列

<br/>

  <br/>

### オブジェクトの置換と関数の返却

#### オブジェクトの置換

- **オブジェクト間で置き換え可能**

- 例）

  ~~~c++
  class Circle {
  private:
 int radius;
  public:
 Circle() {
 radius = 1;
 }
 Circle(int r) {
 radius = r;
 }
 int getRadius() {
 return radius;
 }
  };
  
  
  int main()
  {
 Circle c1(5);
 Circle c2(30);
 cout << "置換前の c1 の半径 :" << c1.getRadius() << endl;
 cout << "置換前の c2 の半径 : " << c2.getRadius() << endl;
      c1 = c2;
 cout << "入れ替え後の c1 の半径 : " << c1.getRadius() << endl;
 cout << "入れ替え後の c2 の半径 : " << c2.getRadius() << endl;
  }
  ~~~

  **結果**

  ~~~
  置換前のc1半径：5
  置換前のc2半径：30
  置換後のc1半径：30
  置換後のc2半径：30
  ~~~

<br/>

- サンプルコードでは、c1の半径の値は5、c2は30となっている。

  置換前は、c1とc2の半径の値がそれぞれ生成され、初期値である5と30の値を持つ。

  c1=c2; を用いて c1 に c2 の値を代入した後、c1 は c2 と同様に 30 という値を持つ。

<br/>

#### オブジェクトの返却

- **オブジェクトの関数返却**
- 例

~~~c++
class Circle {
private:
    int radius;
public:
    Circle() {
    radius = 1;
    }
    Circle(int r) {
    radius = r;
    }
    int getRadius() {
    return radius;
    }
};

Circle getCircle() {
    Circle tmpCircle(30);
    return tmpCircle;
}

int main()
{
    Circle c1(5);
    cout << "オブジェクトの返却前のc1の半径 >>" << c1.getRadius() << endl;
    c1 = getCircle();
    cout << "オブジェクトの返却後のc1の半径 >>" << c1.getRadius() << endl;
}
~~~

**結果**

~~~
オブジェクトを返す前：c1の半径 >>5
オブジェクトを返した後：c1の半径 >>30
~~~

<br/>

- サンプルコードには、値が5のc1オブジェクトとgetCircleオブジェクトが存在する。

getCircle オブジェクトは、内部に半径が 30 の tmpCircle オブジェクトを返す

​  c1 = getCircle() を使用して c1 に getCircle オブジェクトの関数を代入すると、c1 は getCircle オブジェクト内の tmpCircle の半径値である 30 を持つことになる。

<br/>

## 参照

### 変数の参照

- 形式: **&変数名 = 参照変数名;**

- 例）

  ~~~C++
  int main(){
 
 int a = 20;
  int &refv = a;
 
 cout << "a >>" << a << endl; 
 cout << "refv >>" << refv << endl;
  }
  ~~~

  **結果**

  ~~~
  a >>20
  refv >>20
  ~~~

  <br/>

  - 変数 a は 20 として宣言され、初期化される。

    refvは変数aを参照して生成され、値は20に初期化される。

  <br/>

### オブジェクトの参照

~~~c++
class Circle {
private:
    int radius;
public:
    Circle() {
 radius = 1;
    }
    Circle(int r) {
 radius = r;
    }
    int getRadius() {
 return radius;
    }
    int setRadius(int radius) {
 return this->radius = radius;
    }
};


int main()
{
    Circle circle;
    Circle& refCircle = circle;
    refCircle.setRadius(20);
    cout << "circleの半径: " << circle.getRadius() << endl;
    cout << "refCircleの半径: " << refCircle.getRadius() << endl;
}
~~~

**結果**

~~~
円の半径：20
参照円の半径：20
~~~

<br/>

- refCircleは、同じくCircleオブジェクトであるcircleを参照している。

  circleは引数なしでインスタンス化され、radiusの値は1となる。

  refCircleでsetRadius(20)を実行すると、circleの値も更新され、refCircleとcircleの半径はどちらも20になった。
