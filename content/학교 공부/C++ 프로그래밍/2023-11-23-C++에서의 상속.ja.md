## 相続

### 継承されたオブジェクトへのアクセス

```c++
class Shape {
    int x, y; //한 점 (x,y)의 좌표값
    int type; //0:미지정, 1:원 2:사각형
public:
    Shape() { type = 0; }
    void set(int x, int y) {
        this->x = x;
        this->y = y;
    };
    void showShape() {
        cout << "(" << x << "," << y << ")" << endl;
    }
};
    class Circle : public Shape {
        string color;
    public:
        void setColor(string color) { 
            this->color = color; 
        }
        void showCircle();
    };
    void Circle::showCircle() {
        cout << color << ":";
        showShape(); //Shape의 showShape() 호출
    }
    
int main()
{
    Circle a;
    a.set(1, 2); //기본 클래스 Shape의 set(int x, int y) 호출
    a.showShape(); //기본 클래스 Shape의 showShape() 호출
    a.setColor("Red"); //파생 클래스 Circle의 setColor(string color) 호출
    a.showCircle(); //파생 클래스의 showCircle();
}
    
```

**出力**

```
(1,2)
Red:(1,2)
```

<br/>

- CircleクラスはShapeクラスを継承する。

  

  - 派生クラスのオブジェクトであるaにおいて、set()とshowShape()は親クラスから継承した関数を呼び出し、

  - setColor(string color) と showCircle() は派生クラスで実装されているため、派生クラス内の関数を呼び出す。

    <br/>

### 継承関係のアクセス指定子

- #### 非公開

  - 宣言されたクラス内部からのみアクセス可能

   - 派生クラスからは直接アクセスできない

- #### 公開

  - 宣言されたクラスや外部の任意のクラスから、すべての外部関数へのアクセスを許可
  - 派生クラスから基底クラスのpublicメンバーにアクセス可能

- #### 保護済み

  - 宣言されたクラスからアクセス可能
  - 派生クラスからのみアクセスを許可

#### 例

```c++
using namespace std;

#include <iostream>

    cout << color << ':';
```

### コンストラクタとデストラクタの呼び出し

- #### コンストラクタの実行

  - 基本クラスのコンストラクタを先に実行する

- #### 消滅者の実行

  - 派生クラスのデストラクタが先に実行される

```c++
class Parent_Constructor {
public:
    Parent_Constructor() {
        cout << "부모 클래스에서 생성자가 호출 되었습니다." << endl;
    }
    ~Parent_Constructor() {
        cout << "부모 클래스에서 소멸자가 호출 되었습니다." << endl;
    }
};
class Child_Constructor : public Parent_Constructor {
public:
    Child_Constructor() {
        cout << "자식 클래스에서 생성자가 호출 되었습니다." << endl;
    }
    ~Child_Constructor() {
        cout << "자식 클래스에서 소멸자가 호출 되었습니다." << endl;
    }
};


int main(){
    Child_Constructor cc;
}
```

**出力**

```
부모 클래스에서 생성자가 호출 되었습니다.
자식 클래스에서 생성자가 호출 되었습니다.
자식 클래스에서 소멸자가 호출 되었습니다.
부모 클래스에서 소멸자가 호출 되었습니다.
```

- 上述のように、親クラスと子クラスの両方がコンストラクタを持っている場合、コンストラクタは親クラス→子クラスの順に実行される
- 親クラスと子クラスの両方がデストラクタを持っている場合、逆に子クラス→親クラスの順にデストラクタが実行される

### 相続人の指定

- #### 相続指定者

  - ##### 公開

    - 基本クラスのprotectedおよびpublicメンバー変数を**そのまま**継承

  - ##### 非公開

    - 基底クラスのprotectedおよびpublicメンバーを**private**として継承

  - ##### 保護済み

    - 基底クラスのprotectedおよびpublicメンバーを**protected**として継承



### 多重継承

- 複数のクラスを同時に継承すること
- 例）

	```c++
	class multimediaPlayer : public mp3Player, public videoPlayer
	```
	
	

<br/>

##### サンプルコード

```c++
//다중 상속
class Adder { protected: int add(int a, int b) { return a + b; } };
class Subtractor { protected: int minus(int a, int b) { return a - b; } };

class Calculator : public Adder, public Subtractor { public: int calc(char op, int a, int b); };
int Calculator::calc(char op, int a, int b) {
    int res = 0;
    switch (op){
        case '+': res = add(a,b); break;
        case '-': res = minus(a, b); break;
        }
    return res;
}


int main()
{
    //다중 상속
    Calculator handCalculator;
    cout << " 2 + 4 = " << handCalculator.calc('+', 2, 4) << endl;
    cout << "100 - 8 = " << handCalculator.calc('-', 100, 8) << endl;
}
```

```
Output:

 2 + 4 = 6
100 - 8 = 92
```

### 派生クラスでの関数の再定義

```c++
class Base {
public:
    void f() { cout << "Base::f() called" << endl; }
};

class Derived : public Base {
public:
    void f() { cout << "Derived::f() called" << endl; }
};


int main(){
	Derived d, * pDer;
    pDer = &d;
    pDer->f(); //Derived::f() 호출

    Base* pBase;
    pBase = pDer; //업캐스팅
    pBase->f(); //Base::f() 호출
}
```

**出力**

```
Derived::f() called
Base::f() called
```



### 仮想関数とオーバーライド

#### 仮想関数

- **Virtual**として宣言
  - 関数への呼び出しのバインディングを実行時に決定（動的バインディング）
- オーバーライド
  - 派生クラスで、基底クラスの関数と同じ名前の関数を宣言
    - 基底クラスの関数の内容を再定義
    - 多態性を実現するための機能
    - 仮想関数をオーバーライドする場合、動的バインディングが可能

```c++
class Base2 { 
    public: 
        virtual void f() {  //가상 함수
            cout << "Base::f() Called" << endl; 
        } 
};
class Derived2 : public Base2 { 
public: 
    virtual void f() {  //오버라이딩
        cout << "Derived::f() called" << endl; 
    } 
};


int main(){
    Derived2 d, * pDer2;
    pDer2 = &d;
    pDer2->f(); //Derived::f() 호출

    Base2* pBase2;
    pBase2 = pDer2; //업캐스팅
    pBase2->f(); //Base::f() 호출
}
```

**出力**

```
Derived::f() called
Derived::f() called
```

<br/>

- 静的バインディング

  - 関数の呼び出しをコンパイル時に決定すること

- 動的バインディング

  - 関数の呼び出しを実行時に決定すること

  ![画像の代替テキスト]({{link}}/assets/img/Cpp/11th/Binding_Type.png )

**例**

```C++
class Base2 { 
    public: 
        virtual void f() { 
            cout << "Base::f() Called" << endl; 
        } 
};
class Derived2 : public Base2 { 
public: 
    virtual void f() { 
        cout << "Derived::f() called" << endl; 
    } 
};


int main(){
    Derived2 d, * pder; //Base 멤버의 void f()는 호출되지 않음
    pder = &d;
    pder->f(); //derived::f() 호출

    Base2* pbase;
    pbase = pder; //업 캐스팅
    pbase->f(); // 동적 바인딩 발생 derived::f() 실행
}
```

**出力**

```
Derived::f() called
Derived::f() called
```
