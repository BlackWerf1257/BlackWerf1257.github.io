## 消滅者

### 意味

オブジェクトが破棄される際に自動的に呼び出される関数

<br/>

### 特徴

- 任意に呼び出すことはできません

- 2つ以上存在することはできない（多型化は不可能）

- オブジェクトが破棄される際、割り当てられたメモリの解放、ファイルの保存および閉じ、ネットワークの切断などの処理を行う

- クラス名の前に**~**を付けて使用する

  ​  例

  ~~~c++
  class Circle{
 Circle();
 ~Circle(); 
 }
  Circle::~Circle();
  ~~~

  

  

- 戻り値がない

- 宣言されていない場合、デフォルトのデストラクタが自動的に生成される



### コンストラクタとデストラクタの実行順序

#### オブジェクトの生成位置による分類

##### 地域オブジェクト

- 関数内で宣言されたオブジェクトは、関数が終了した際に破棄される

<br/>

##### グローバルオブジェクト

- 関数の外部で宣言されたオブジェクトは、プログラムの終了時に破棄される

<br/>

#### オブジェクトの生成順序

- グローバルオブジェクト：プログラム内で宣言された順序で生成される
- ローカルオブジェクト：関数が呼び出される瞬間に、その順序で生成される

<br/>

#### オブジェクトの破棄順序

- 関数の終了時、ローカルオブジェクトは生成順の逆順で破棄される
- プログラムの終了時、グローバルオブジェクトは生成順の逆順で破棄される

<br/>

#### new キーワードで動的に生成されたオブジェクト

- `new`が実行された瞬間にオブジェクトが生成される
- `delete`演算子が実行された瞬間にオブジェクトが破棄される

**サンプルコード**

```c++
class Circle {
	public:
		int radius;

		Circle();
		Circle(int r);
		~Circle(); //소멸자
		double getArea();
};
Circle::Circle() {
	radius = 5;
	cout << "지름이 " << radius * 2 << "인 원 생성" << endl;
}
Circle::Circle(int r) {
	radius = r;
	cout << "지름이 " << r * 2 << "인 원 생성" << endl;
}
Circle::~Circle() {
	cout << "지름이 " << radius * 2 << "인 원 소멸" << endl;
}
double Circle::getArea() {
	return 3.14 * radius * radius;
}

Circle globalDonut(1000);
Circle globalPizza(2000);
void func() {
	Circle fDonut(100);
	Circle fPizza(200);
}
	
	
	
int main(){
	Circle mainCircle1;
    Circle mainCircle2(10);
    func();
}
```

**結果**

```

지름이 4000인 원 생성
지름이 10인 원 생성
지름이 20인 원 생성
지름이 200인 원 생성
지름이 400인 원 생성
지름이 400인 원 소멸
지름이 200인 원 소멸
지름이 20인 원 소멸
지름이 10인 원 소멸
지름이 4000인 원 소멸
지름이 2000인 원 소멸
```

<br/>

**サンプルコードの実行順序**

1. **プログラムの読み込み** - globalDonut、globalPizza オブジェクトの生成
2. **main 関数の実行** - mainCircle1、mainCircle2 オブジェクトの生成
3. **func関数の実行** - fDonut、fPizzaオブジェクトの生成
4. **func関数の終了** - fDonut、fPizzaオブジェクトの破棄
5. **main関数の終了** - mainCircle1、mainCircle2オブジェクトの破棄
6. **プログラムの終了** - globalDonut、globalPizzaオブジェクトの破棄

## アクセス指定子

### 種類

- **public** - 外部の関数からアクセス可能
- **private（デフォルト）** - 同一クラスのメンバ関数からのみアクセス可能
- **protected** - 自身および派生クラスからのみアクセス可能

### 特徴

- オブジェクトのカプセル化をサポートする
- メンバーへのアクセス権限を指定する

<br/>

## インライン関数

### 関数呼び出しプロセスの問題点：

- 頻繁な呼び出しは、不要なオーバーヘッドの問題を引き起こす

  スタックにレジスタを保存 -> 関数の呼び出しと実行 -> スタックに保存されたレジスタの復元 -> 関数の戻り

  レジスタ - リターンアドレス、ローカル変数の値、引数

### 使い方

- **inline** キーワードの使用

- プリプロセッサによって、コンパイル時に自動的にコードに挿入される

- サンプルコード

  ~~~c++
  inline int divide(int x) {
 return (x/2);
  }
 
 int main() {
    int sum = 0;
 
 for(int i=1; i<=3; i++) {
 if(divide(i)) //(x/2) を呼び出す
  		{
 cout << i << " が呼び出されました " << endl;
 sum += i;
 }
    }
    cout << "----" << endl;
    cout << sum;
  }
  ~~~

  **結果**

  ~~~
  2 呼び出し 
  3 呼び出し 
  ----
  5
  ~~~

  <br/>

### メリット

- 頻繁に呼び出される関数のオーバーヘッドを削減 -> パフォーマンスの向上

### デメリット

- 実行プログラムのサイズ増加によるメモリオーバーヘッド

### 自動インライン関数

- inlineキーワードを使用しなくても、コンパイラがインライン展開を行う関数

- 例）

  ~~~c++
  class Circle{
  private:
 int radius;
  public:
 Circle() { radius = 1; } //自動インライン関数
 Circle(int r);
 double getArea() { return 3.14 * radius * radius; }
  };
  ~~~

## 構造体

### 特徴

- クラスと類似（継承、メンバー、アクセス指定子）
- デフォルトのアクセス指定子は **public**
- C言語と互換性がある

### 使い方

~~~c++
struct 構造体名{
  型 メンバー変数1;
    ...
  型 メンバー変数n;
}
~~~

<br/>

**サンプルコード**

~~~c++
struct Circle {
  private:
    int radius;
  public:
    Circle(int r) { radius = r; } //構造体のコンストラクタ
    double getArea();
};
double Circle::getArea() {
  return 3.14 * radius * radius;
}


int main(){
    Circle donut(5);
    cout << "面積は " << donut.getArea();
}
~~~

**結果**

~~~
面積は78.5
~~~



## ファイルの分割

### 利用理由

**クラスの宣言と実装を分離する**

**宣言と実装の分離**

- クラスの再利用や修正が効率的
- クラスの宣言部はヘッダーファイルで管理する
- クラスの実装部はCPPファイルで管理する

<br/>

**プロジェクト管理に効率的**

- 開発業務の分担に直感的に対応
- クラス単位で細分化されたコードを管理
- 共同作業の際、変更による影響を最小限に抑える

<br/>

### 外部ファイルの使用方法

<br/>

## オブジェクトポインタ

- オブジェクトのアドレスを持つ変数

- ポインタを介してメンバーにアクセスする場合、**p->メンバー**という形式でアクセスする

  <br/>

### 使い方

- ポインタ変数の宣言
  - **クラス名 *変数**
- ポインタ変数へのアドレスの指定
  - **&オブジェクト名**
- オブジェクトポインタによるメンバへのアクセス
  - **->**

<br/>

**サンプルコード**

~~~c++
class CirclePointer {
  int radius;
  public:
    CirclePointer() { radius = 1; }
    CirclePointer(int r) { radius = r; }
    double getArea();
};
double CirclePointer::getArea() {
  return 3.14 * radius * radius;
  }


int main(){
// オブジェクトポインタ
CirclePointer donut;
CirclePointer pizza(30);

//オブジェクト名を使ってメンバーにアクセス
cout << donut.getArea() << endl;

//オブジェクトポインタによるメンバへのアクセス
CirclePointer *p;
p = &donut;
cout << p->getArea() << endl; //donutのgetAreaを呼び出す
cout << (*p).getArea() << endl; //donutのgetAreaを呼び出す

p = &pizza;
cout << p->getArea() << endl; // pizzaのgetAreaを呼び出す
cout << (*p).getArea() << endl; // pizzaのgetAreaを呼び出す
}
~~~

**結果**

~~~
3.14
3.14
3.14
2826
2826
~~~
