## 関数の重複を解消する

### デフォルト引数を用いた関数の重複を簡素化

サンプルコード

```c++
Circle() { radius = 1; }
Circle(int r) { radius = r; }

			||

Circle(int r=1) { radius = r; }
```

```
void drawSymbol(){
	for(int i=0; i<10; i++) cout << '*';
	cout << endl;
}
void drawSymbol(int n, char c){
	for(int i=0; i<n; i++) cout << c;
	cout << endl;
}


				||
			
void drawSymbol(int n=10; char c='*'){
	for(int i=0; i<n; i++) cout << c;
	cout << endl;
}
```



- デフォルト引数は、関数の重複と関数の区別とは関係がない

<br>

### 関数の重複による曖昧さ

#### 正常にコンパイル

```C++
double square(double a) { return a*a; }

int main() { cout << square(3); }
```



#### 変数の型変換による関数の重複に関する曖昧さ

```c++
float square(float a) {return a*a;}
double square(double a) { return a*a; }

int main() { return square(3) }
```



#### デフォルト引数による関数の重複がもたらす曖昧さ

```c++
 void msg(int id) { cout << id << endl;  }
 void msg(int id, string s= "") { cout << id << ":" << s << endl; }

 int main() { msg(5); }
```



#### 参照パラメータによる関数の重複に伴う曖昧さ

```c++
int add(int a, int b)    { return a + b; }
int add(int a, int &b) { return b = b + a; }

int main(){
	int a=10, b=20;
	cout << add(a,b,);
}
```

## 静的

### 定義

* 変数の宣言
* プログラムの開始時に生成され、終了時に破棄される

- クラスの静的メンバー
  - 静的メンバー
 * プログラムの開始時に生成される
 * クラスごとに1つだけ生成される（すべてのオブジェクトが1つのメンバーを共有する）
  - 宣言
    * static 型 変数名;
    * 例) static int totalAccount;
  - 初期化
 * 静的メンバー変数に対して、クラス外部での初期化が必要
 * 例) int Bank::totalAccount = 0;

<br/>

**サンプルコード**

```C++
class Bank { public : static int totalAccount; };
int Bank::toatalAccount = 0;
```

### Staticの活用

- オブジェクト間の共有
  * スタティックメンバーを宣言したクラスから生成されたすべてのオブジェクト間で共有される

* 制約事項
  * staticメンバーがpublicとして宣言されている場合
    * クラスのカプセル化を維持できない
    * 外部からstaticとして宣言されたメンバーにアクセス可能
  * staticメンバー関数へのアクセス
    * staticメンバー関数はnon-staticメンバーにアクセスできない
    * non-static関数はstaticメンバーにアクセス可能



**サンプルコード 3**

```
class Math {
	public:
		static int abs(int a) { return a > 0 ? a : -a; }
		static int max(int a, int b) { return a > b ? a : b; }
		static int min(int a, int b) { return a > b ? b : a; }
};

int main(){
	cout << Math::abs(-1) << endl;
	cout << Math::max(10, 20) << endl;
	cout << Math::min(-5, 5) << endl;
}
```

出力

```
1
20
-5
```

## 友達

### フレンド関数

- クラスのメンバではない外部関数
- クラスのすべてのメンバにアクセスできる権限を持つ
- 種類
  * グローバル関数：クラスの外部で宣言されたグローバル関数
  * 他のクラスのメンバ関数
  * 他のクラスのすべてのメンバ関数

<br/>

**サンプルコード 5 - 外部グローバル関数**

```
class Rectangle;
bool equals(Rectangle r, Rectangle s);
class Rectangle {
	int width, height;
public:
	Rectangle(int width, int height) { this->width = width; this->height = height; }
	friend bool equals(Rectangle r, Rectangle s);
};
bool equals(Rectangle r, Rectangle s) {
	if (r.width == s.width && r.height == s.height)
		return true;
	else
		return false;
}

int main(){
	Rectangle a(1, 2), b(3, 4);
	if (equals(a, b))
		cout << "equal" << endl;
	else
		cout << "not equal" << endl;
}
```

出力

```
not equal
```

## 演算子の重複

### 一般的な演算子の対象

- 数値のみ可能
- 文字列不可
- オブジェクト不可

### C++の演算子の重畳対象

* オブジェクトが可能
* 文字列が可能
* 追加の演算子の再定義が可能



### 重複可能な演算子の種類

| :--: | :--: | :--: | :--: | :----: | :---: | :------: |

### 二項演算子「+」の重複

c = a + b

c = a + b;



**Circle operator+(Circle c1);**



```c++
class Circle {
    int radius;

 public:
```

**関数の実装**

```c++
Circle Circle::operator+(Circle c1){
    Circle tmp;
    tmp.radius = this->radius + c1.radius;
    tmp.price = this->price + c1.price;
    return tmp;
}
```

関数の実装スクリプトにおいて、this は a. オブジェクトを意味する
