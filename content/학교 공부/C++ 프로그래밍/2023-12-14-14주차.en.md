Map

### Features

- A generic container that stores pairs of 'key' and 'value' as elements
  - An error occurs if data with the same 'key' is stored more than once
- Required in many applications
- To use it, you must declare **'#include<map>'**

### Functions

| Function | Description |
| -------------------------- | ---------------------------------------------- |
| insert(pair<> *element)    | Inserts a pair object consisting of a key and a value |
| at(key_type& key) | Returns the value corresponding to the key |
| begin() | Returns a reference to the first pair object |
| end() | Returns a reference pointing to the end of the map |
| empty() | Returns true if the map is empty |
| find(key_type& key) | Returns an iterator pointing to the element corresponding to the key |
| erase(key_type& key) | Deletes the element pointed to by the iterator |
| size() | Returns the number of elements in the map |
| operator[key_type& key] () | Returns the value associated with the key |
| operator=() | Copies this map to another map |



### Example

```c++
int main()
{
    map<string, string> dic;
    dic.insert(make_pair("love", "사랑"));
    dic.insert(make_pair("apple", "사과"));

    dic["cherrry"] = "체리"; //("cherry", "체리") 저장
    cout << "사전의 단어 수 : " << dic.size() << endl;

    string eng;
    while (true)
    {
        cout << "찾고 싶은 단어 :";
        getline(cin, eng); //사용자로부터 단어 입력

        if (eng == "exit")
            break;
        if (dic.find(eng) == dic.end()) //해당 키가 존재 X할 경우
            cout << "값 존재 X" << endl;
        else
            cout << dic[eng] << endl;
    }
    cout << "종료" << endl;
}
```

**Output**

```

찾고 싶은 단어 :apple
사과
찾고 싶은 단어 :ex
값 존재 X
찾고 싶은 단어 :exit
종료
```

 ## Stream

  - Data Flow
  - Software modules that transmit data
  - **Programs and devices are connected at both ends of the stream**
    - Data is transmitted in the order it was sent
    - Basic unit of input/output: Byte
  
  
  
  ### Types
 
 - Input Stream
    - A stream that delivers data from input devices, networks, or files to a program

  - Output stream
    - A stream that transmits data output by a program to an output device, a network, or a file
  

  <br/>

  ### Input and Output Streams

  - The C++ standard supports only the stream-based approach

  - Streams in the C++ standard have buffers
 
 - Input stream buffer
 
 - Purpose: 
 1. Temporarily stores data entered from an input device before passing it to the program
 2. Allows for modifications during keystrokes
 - When the BackSpace key is pressed, the previously entered character is deleted from the buffer
  
    - **The program reads the values after the user has finished entering input**
 
 - Enter key: Signifies the end of key input
 
 - From the moment this key is pressed, the program begins reading values from the key input buffer
 
 <br/>
  
      - **Output Stream Buffer**
 
 - Purpose
 - Temporarily stores data output by the program before transmitting it to the output device
 - Improves efficiency by avoiding repeated use of the output device
 - **Data is sent to the output device when the buffer is full or upon a forced output command**
  
  
  
  ### Standard Input/Output Streams
 
 - cin: of type istream, connected to the keyboard
 
 - cout: of type ostream, connected to the screen
 
 - cerr: of type ostream, connected to the screen (stream buffer - X, error output)
  
  - clog: of type ostream, connected to the screen (stream buffer - O, error output)
 
 <br/>
 
 |    Category     | Class | Description |
  | :---------: | :---------------------------------: | :----------------------------------------------------------: |
  | Base Streams | ios | Base class for input/output classes<br/>Declarations of common functions and constant variables for stream classes |
  | Character Streams | istream<br/>ostream<br/>iostream  | istream - Input stream<br/>ostream - Output stream<br/>iostream - Input/output stream |
  | File Streams | ifstream<br/>ofstream<br/>fstream | ifstream - Used to read data from a file<br/>ofstream - Used to write data to a file<br/>fstream - Used to read and write data to a file |

## Format Input/Output

### 1. Format Flags

- Flags in the ios class for specifying input/output formats in input/output streams

  | Flag | Value     | Meaning |
  | --------------- | ------ | ------------------------------------------------------------ |
  | ios::skipws     | 0x0001 | Ignore whitespace characters during input |
  | ios::unitbuf    | 0x0002 | Output data immediately without buffering it in the output stream   |
  | ios::uppercase   | 0x0004 | Print hexadecimal characters A–F and the exponent E in uppercase |
  | ios::showcase   | 0x0008 | Prefix numbers with 0x for hexadecimal or 0 for octal |
  | ios::showpoint  | 0x0010 | For floating-point values, output the integer part along with the trailing zeros after the decimal point |
  | ios::showpos     | 0x0020 | Output a + sign for positive numbers |
  | ios::left | 0x0040 | Output the field left-aligned |
  | ios::right | 0x0080 | Output the field right-aligned |
  | ios::internal   | 0x0100 | Output the sign left-aligned and the number right-aligned |
  | ios::dec | 0x0200 | Output in decimal format and set as default |
  | ios::oct | 0x0400 | Output in octal |
  | ios::hex | 0x0800 | Output in hexadecimal |
  | ios::scientific | 0x1000 | Output floating-point numbers according to scientific notation rules |
  | ios::fixed | 0x2000 | Output floating-point numbers in fixed-point format |
  | ios::boolalpha  | 0x4000 | If enabled, outputs "true" for true and "false" for false<br/>. If not enabled, outputs the integers 1 and 0 |

  <br/>

  - Member function for setting flags

    - long setf(long flags)
 - Sets the flags as the stream's format flags and returns the previous flags
 
 - long unsetf(long flags)
 - Clears the bits set in flags from the stream's format flags and returns the previous flags
 
 **Example**
    

## Exception Handling

```c++
int main()
{
    //예외 처리
    int n, sum, average;
    while (true)
    {
        cout << "합을 입력하세요 : ";
        cin >> sum;

            if (n <= 0) //오류 탐지

            average = 0;
            cout << endl;
            continue;

    }
}
```
