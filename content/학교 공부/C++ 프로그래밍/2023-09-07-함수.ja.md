### コマンド

- 例）

  ```c++
  #include <string.h>
  int n = strlen("test")
  ```

  - **strcmp()**
    - 文字列を比較する関数
    - 使い方: **strcmp(string 値1, string 値2)**
    - 結果:
 - 値が0より大きい場合：string1がstring2より大きい
 - 値が0の場合：string1 = string2
      - 値が0より小さい場合：string1がstring2より小さい
  - **strlen()**
    - 文字列の長さを返す関数
    - 使い方：**strlen(string 値)**
    - 結果：引数として渡されたstring値の長さを返す
  - **strcpy()**
    - 文字列をコピーする関数
    - 使用方法: **strcpy(string1, string2)**
    - 結果: string2の値をstring1にコピーする

#### 入力

食べる

#### 出力

#### サンプルコード

```c++
#include <iostream>
using namespace std;

int main(){
    cout << "글자를 입력하세요>>";
    char name[11]; //한글 - 5글자 | 영문 - 10문자
    cin >> name;
    
    cout << "출력된 문자 >> " << name << endl;
}
```

**結果**

~~~
文字を入力してください>>Arnold 
表示された文字 >> Arnold
~~~

<br/>

#### 空白を含む文字列の入力

-  **cin.getline(char buf[], int size, char delimitChar)**

  - bufにsize-1サイズの文字を入力する

  - delimitChar に遭遇した場合、入力が中断される
    - delimitChar と buf の末尾には、いずれも '\0' が付加される
  - delimitChar のデフォルト値は **'\n'(Enter)**

  <br/>

  - #### サンプルコード

    ```c++
    #include <iostream>
    using namespace std;
    
    int main(){
        cout << "글자를 입력하세요>>";
        char name[20];
    	cin.getline(name, 100, '\n');
        
        cout << "출력된 문자 >> " << name << endl;
    }
    ```

    **結果**

    ~~~
    文字を入力してください>>Arnold Michael 
 出力された文字 >> Arnold Michael 
 ~~~

### String クラス

- Charとは異なり、文字列の長さに制限がない
- 文字列のコピーや比較など、さまざまな関数や演算子を提供している
- オブジェクト指向である
- Stringヘッダーをインクルードする必要がある



#### サンプルコード

```c++
#include <iostream>
#include <string>
using namespace std;

int main(){
    string University("백석대학교");
    string Department("첨단IT 학부");
    string Major("AR/VR");
    
    cout << "현재 " << University << " " << Department << " " << Major << "전공으로 재학중입니다 " << endl;
} 
```

**結果**

```
현재 백석대학교 첨단IT 학부 AR/VR전공으로 재학중입니다
```
