## Simplifying Duplicate Functions

```c++

			||
```

## Static

### Definition

* Declaration of variables
* Created at the start of the program and destroyed when the program ends

- Static members of a class
  - Static members
 * Created when the program starts
 * Only one is created per class (all objects share a single member)
  - Declaration
    * static variable_type variable_name;
    * Ex) static int totalAccount;
  - Initialization
 * External initialization of static member variables is required
 * Ex) int Bank::totalAccount = 0;

<br/>

**Sample Code**

```C++
class Bank { public : static int totalAccount; };
int Bank::toatalAccount = 0;
```

### Accessing Static Members Using the Scope Resolution Operator (::)

*  Since only one static member is created per class, it can be accessed using the **class name and scope resolution operator**



* Format

  **Class Name:: Static Member**

  ```
  kim.sharedMoney = 200;
  lee.addShared(200);
  
  		||
  
  Person::sharedMoney = 200;
  Person::addShared(200);
  ```



### Using Static

- Shared among objects
  * Shared by all objects created from the class in which the static member is declared

* Limitations
  * When a static member is declared as public
    * Class encapsulation cannot be maintained
    * External code can access the static member
  * Access to static member functions
    * Static member functions cannot access non-static members
    * Non-static functions can access static members



**Example Code 3**

## Friend

### Friend Functions

- External functions that are not members of the class
- Have permission to access all members of the class
- Types
  * Global functions: Global functions declared outside the class
  * Member functions of other classes
  * All member functions of other classes

<br/>

**Example Code 5 - External Global Functions**

```
class Rectangle;
bool equals(Rectangle r, Rectangle s);
class Rectangle {
	int width, height;
public:
	Rectangle(int width, int height) { this->width = width; this->height = height; }
	friend bool equals(Rectangle r, Rectangle s);
};
bool equals(Rectangle r, Rectangle s) {
	if (r.width == s.width && r.height == s.height)
		return true;
	else
		return false;
}

int main(){
	Rectangle a(1, 2), b(3, 4);
	if (equals(a, b))
		cout << "equal" << endl;
	else
		cout << "not equal" << endl;
}
```

Output

```
not equal
```

### Duplication of the Binary Operator +

c = a + b

c = a. + (b);



**Circle operator+(Circle c1);**



```c++
class Circle {
    int radius;
    int price;
 public:
    ...
	    Circle operator+(Circle c1);
}
```

**Implementing the Function**

```c++
Circle Circle::operator+(Circle c1){
    Circle tmp;
    tmp.radius = this->radius + c1.radius;
    tmp.price = this->price + c1.price;
    return tmp;
}
```

In the function's implementation script, `this` refers to the `a` object.

```C++
class Circle9 {
```
