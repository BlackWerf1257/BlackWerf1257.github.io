## 動的バインディング

### 特徴

- プログラムの実行中に、基底クラスのポインタを使用してオーバーライドされた派生クラスの関数を実行する
- 基底クラスの関数をVirtualとして宣言する
- オーバーロードでは使用不可（オーバーライドでのみ使用可能）



###  仮想関数のオーバーライド

#### 特徴

- 関数名、引数の型、引数の数、戻り値の型が一致している

- 派生クラスではvirtualを省略可能（継承されるため）

**例**

```c++
class Base{ public: virtual void f(); };
class Derived : public Base { public: virtual void f(); } //vrtual void f()와 동일한 선언으로 상속되었으므로 virtual 생략가능
```

**サンプルコード 1**

```c++
class Base {
    public: 
    virtual void f() 
    { cout << "Base::f() Called" << endl; }; 
};
class Derived : public Base {
    public: 
    void f(){
        cout << "Derived::f() Called" << endl; 
    }; 
};
class AgainDerived : public Derived { 
    public: 
    void f() { 
        cout << "AgainDerived:f() called" << endl; 
    } 
};

int main()
{
    AgainDerived a;
    Base* bp;
    Derived *dp;
    AgainDerived* ap;
    
    bp = dp = ap = &a;

    bp->f();
    dp->f();
    ap->f();
}

```

**出力**

```
AgainDerived:f() called
AgainDerived:f() called
AgainDerived:f() called
```



### オーバーライドのスコープ指定演算子（::）

- スコープ指定演算子 **'::'**
  - 静的バインディングの指定
  - **基底クラス::仮想関数()** の形式で、基底クラスの仮想関数を静的バインディングにより呼び出す
    - 例）Shape::draw();



## 仮想消滅者

- デストラクタをVirtualキーワードで宣言
- デストラクタの呼び出し時に動的バインディングが発生
  - 派生クラスのデストラクタが優先される



#### 仮想消滅子の場合

```c++
class Base4 {
public:
    virtual ~Base4() { cout << "Called from Base" << endl; };
};
class Derived4 : public Base4 {
public:
    ~Derived4() { cout << "Called from Derived" << endl; };
};


int main(){
    Base4* pT = new Derived4();
	delete pT;
}
```

**出力**

```
Called from Derived
Called from Base
```



1. ~Base4() のデストラクタを呼び出す
2. ~Derived4() を実行する
3. ~Base4() を実行する

## 仮想関数の活用

- 仮想関数を用いた動的バインディングの例
  - 基底クラスのポインタを通じて派生クラスにアクセス
  - pStart、pLast、pの型がShape*
  - リンクリストをたどって、Shapeを継承した派生オブジェクトにアクセス
  - p->paint()の呼び出しにより、派生オブジェクトでオーバーライドされたdraw関数を呼び出す



```c++
#include "Shape.h"
#include "Circle.h"
#include "Rect.h"
#include "Line.h"

int main(){
    Shape* pStart = NULL;
Shape* pLast;

pStart = new Circle();
pLast = pStart;
pLast = pLast->add(new Rect());
pLast = pLast->add(new Circle());
pLast = pLast->add(new Line());
pLast = pLast->add(new Rect());

//현재 연결된 모든 도형 그리기
Shape* p = pStart;
while (p != NULL) {
    p->paint();
    p = p->getNext();
}
//현재 연결된 모든 도형 삭제
p = pStart;
while (p != NULL) {
    Shape* q = p->getNext(); //다음 도형 주소 기억
    delete p; //기본 클래스의 가상 소멸자 호출
    p = q; //다음 도형 주소를 p에 저장
	}
}
```

**出力**

```
Circle
Rectangle
Circle
Line
Rectangle
```





## 純粋な仮想関数

- 基底クラスの仮想関数の目的

  - 派生クラスで再定義すべき関数を指定する役割

    - 実行するコードを作成する目的ではない

  - 基底クラスの仮想関数を実装する必要はない

    

- 純粋な仮想関数

  - 純粋仮想関数

  - **コードがXで、宣言のみ存在する仮想メンバー関数**

    - 不要なコードの重複を排除可能

      

  ### 宣言方法

```c++
class Shape{
public:
	 	virtual void draw() = 0; //순수 가상 함수 선언
}
```





## 抽象クラス

### 特徴

- 1つ以上の純粋仮想関数を持つクラス
- 純粋仮想関数があるため、オブジェクトの作成は不可能
  - 継承・設計を目的とする
- ポインタの宣言は可能
  - 動的バインディングを利用した活用が可能



### 相続

- 抽象クラスを単純に継承すると、自動的に抽象クラスになる

### 実装

- 抽象クラスを継承し、純粋仮想関数をオーバーライドする

  - 派生クラスは抽象クラスX

  <br/>

  #### 誤った実装例

```c++
class Shape {
public:
    virtual void draw() = 0;
};
class Circle : public Shape {
public:
    string toString() { return "Circle 객체"; }
};

Shape shape;
Circle waffle;
```

- ShapeとCircleはどちらも抽象クラスです

  - したがって、shapeとwaffleの両方でオブジェクト生成エラーが発生する

    **エラー名**

```
심각도	코드	설명	프로젝트	파일	줄	비표시 오류(Suppression) 상태
오류(활성)	E0322	추상 클래스 형식 "Circle"의 개체를 사용할 수 없습니다.

```



#### 正しい実装方法の例

```c++
class Shape { //추상 클래스
public:
    virtual void draw() = 0;
};
class Circle : public Shape { //추상 클래스가 아님
public:
    virtual void draw() { // 순수 가상 함수 오버라이딩
        cout << "Circle";
    }
    string toString() { return "Circle 객체" };
};

Shape shape;   //추상 클래스이므로 객체 생성 오류 발생
Circle waffle; //순수 가상함수를 오버라이드 했으므로 정상적으로 객체가 생성됨
```





**例**

```c++
class Calculator {
    void input() {
        cout << "정수 2개 입력 >>";
        cin >> a >> b;
    };

protected:
    int a, b;
    virtual int calc(int a, int b) = 0; //두 정수의 합 리턴
public:
    void run() {
        input();
        cout << "계산된 값 : " << calc(a, b) << endl;
    }
};
class Adder : public Calculator {
protected:
    int calc(int a, int b) { //순수 가상 함수 구현
        return a + b;
    }
};
class Subtractor : public Calculator {
protected:
    int calc(int a, int b) { //순수 가상 함수 구현
        return a - b;
    }
};


int main(){
	Adder adder;
Subtractor subtractor;
adder.run();
subtractor.run();    
}
```

**出力**

```
정수 2개 입력 >>  //5 7로 진행
계산된 값 : 12
정수 2개 입력 ->> // 7 5로 진행
계산된 값 : 2
```

- **'+'** → **'-'** の順に進める

## ジェネリック関数の具体化

### 具体化

- コンパイラがテンプレート関数から、データ型に応じた具体化された関数のコードを生成する

**例**

```c++
template <class T>
    void swap(T& a, T& b){
    T tmp;
    tmp = a;
    a = b;
    b = tmp;
}

int main(){
	int a = 4, b = 5;
    swap(a,b);
}
```

### 例 2

```c++
template <class T>
T add(T data[], int n) { //배열 dta에서 n개 원소 합한 결과 리턴
    T sum = 0;
    for (int i = 0; i < n; i++) 
        sum += data[i];

    return sum; //Sum과 타입과 리턴 타입 모두 T로 선언됨

}
```
