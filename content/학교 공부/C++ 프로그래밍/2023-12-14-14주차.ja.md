地図

### 特徴

- 「キー」と「値」のペアを要素として格納するジェネリックコンテナ
  - 同じ「キー」を持つデータを重複して格納するとエラーが発生する
- 多くのアプリケーションで必要とされる
- 使用するには、**'#include<map>'** を宣言する必要がある

### 関数

| 関数 | 説明 |
| -------------------------- | ---------------------------------------------- |
| insert(pair<> *element)    | キーと値で構成されるpairオブジェクトを挿入 |
| at(key_type& key) | キーに対応する値を返す |
| begin() | 最初のpairオブジェクトへの参照を返す |
| end() | mapの末尾を指す参照を返す |
| empty() | mapが空の場合、trueを返す |
| find(key_type& key) | キーに対応する要素を指すイテレータを返す |
| erase(key_type& key) | イテレータが指す要素を削除 |
| size() | mapのデータ数を返す |
| operator[key_type& key] () | キーに対応する値を返す |
| operator=() | このmapを別のmapにコピー |



### 例

```c++
int main()
{
    map<string, string> dic;
    dic.insert(make_pair("love", "사랑"));
    dic.insert(make_pair("apple", "사과"));

    dic["cherrry"] = "체리"; //("cherry", "체리") 저장
    cout << "사전의 단어 수 : " << dic.size() << endl;

    string eng;
    while (true)
    {
        cout << "찾고 싶은 단어 :";
        getline(cin, eng); //사용자로부터 단어 입력

        if (eng == "exit")
            break;
        if (dic.find(eng) == dic.end()) //해당 키가 존재 X할 경우
            cout << "값 존재 X" << endl;
        else
            cout << dic[eng] << endl;
    }
    cout << "종료" << endl;
}
```

**出力**

```
사전의 단어 수 : 3
찾고 싶은 단어 :apple
사과
찾고 싶은 단어 :ex
값 존재 X
찾고 싶은 단어 :exit
종료
```

### sort() 関数の例
 
 - 2つの引数
 
 - 1番目の引数：ソートを開始する要素のアドレス
 
 - 2番目の引数：ソート範囲の最後の要素の次のアドレス
  
    - 3番目の引数：昇順、降順、ユーザー定義（省略可能）
 
 ```c++
      vector<string> vec;
      	...
              
      sort(v.begin(), v.begin()+3) //v.begin() - v.begin()+2까지, 처음 3개 원소 정렬
      ```




### 例
 
 ```c++
  int main()
  {
      //STL 알고리즘 sort()의 예제
      vector<int> v; //정수 벡터 생성
      cout << "5개의 정수 입력하세요 : ";
          for (int i = 0; i < 5; i++)
          {
              int n;
              cin >> n;
              v.push_back(n); //키보드에 읽은 정수를 벡터에 삽입
          }
          //v.begin() - v.end() 사이의 값을 오름차순으로 정렬
          //sort() 함수의 실행결과 : 벡터 v의 원소 순서 변경
          sort(v.begin(), v.end());
          
          vector<int>::iterator it; //벡터 내 원소를 탐색하는 iterator 변수 선언
          for (it = v.begin(); it != v.end(); it++)
              cout << *it << ' ';
          cout << endl;
  }
  ```

 ## Stream

  - データの流れ
  - データを送信するソフトウェアモジュール
  - **ストリームの両端にはプログラムとデバイスが接続**
    - 送信された順序通りにデータが転送される
    - 入出力の基本単位：バイト
  
  
  
  ### 種類
 
 - 入力ストリーム
    - 入力デバイス、ネットワーク、ファイルからプログラムへデータを伝達するストリーム

  - 出力ストリーム
    - プログラムから出力されるデータを、出力デバイス、ネットワーク、またはファイルに転送するストリーム
  

  <br/>

  ### 入出力ストリーム

  - C++標準ではストリーム方式のみをサポートしている

  - C++標準におけるストリームはバッファを持つ
 
 - 入力ストリームのバッファ
 
 - 目的：
 1. 入力デバイスから入力されたデータをプログラムに渡す前に、一時的に保存する
 2. キー入力中に修正が可能である
 - BackSpaceキーが入力されると、以前に入力されたキーがバッファから削除される
  
    - **プログラムは、ユーザーの入力が終了した後に値を読み取る**
 
 - Enterキー：キー入力の終了を意味する
 
 - 当該キーが押された時点から、プログラムはキー入力バッファから値を読み取り始める
 
 <br/>
  
      - **出力ストリームバッファ**
 
 - 目的
 - プログラムから出力されたデータを出力デバイスへ送信する前に、一時的に保存する
 - 出力デバイスを繰り返し使用する非効率性を改善する
 - **バッファがいっぱいになった場合、または強制出力コマンドが発行された場合、出力デバイスへ出力される**
  
  
  
  ### 標準入出力ストリーム
 
 - cin : istream 型で、キーボードに接続
 
 - cout: ostream 型で、画面に接続
 
 - cerr: ostream 型で、画面に接続（ストリームバッファ - X、エラー出力）
  
  - clog: ostream型で、画面と接続（ストリームバッファ - O、エラー出力）
 
 <br/>
 
 |    区分     | クラス | 内容 |
  | :---------: | :---------------------------------: | :----------------------------------------------------------: |
  | 基本ストリーム | ios | 入出力クラスの基底クラス<br/>ストリームクラス共通の関数および定数変数の宣言 |
  | 文字ストリーム | istream<br/>ostream<br/>iostream  | istream - 入力ストリーム<br/>ostream - 出力ストリーム<br/>iostream - 入出力ストリーム |
  | ファイルストリーム | ifstream<br/>ofstream<br/>fstream | ifstream - ファイルからデータを読み込む際に使用<br/>ofstream - ファイルにデータを書き込む際に使用<br/>fstream - ファイルからデータを読み書きする際に使用 |

## フォーマットの入出力

### 1. フォーマットフラグ

- 入出力ストリームで入出力形式を指定するためのiosクラスのフラグ

  | フラグ | 値     | 意味 |
  | --------------- | ------ | ------------------------------------------------------------ |
  | ios::skipws     | 0x0001 | 入力時、空白文字を無視 |
  | ios::unitbuf    | 0x0002 | 出力ストリームに入ってくるデータをバッファリングせず、そのまま出力   |
  | ios::uppercase   | 0x0004 | 16進数のA～F、指数表記のEを大文字で出力 |
  | ios::showcase   | 0x0008 | 16進数なら0x、8進数なら0を数字の前に付けて出力 |
  | ios::showpoint  | 0x0010 | 実数値について、整数部分に加え、小数点以下の末尾の桁を0として出力 |
  | ios:showpos     | 0x0020 | 正の数に対して+記号を出力 |
  | ios::left | 0x0040 | フィールドを左揃え形式で出力 |
  | ios::right | 0x0080 | フィールドを右揃え形式で出力 |
  | ios::internal   | 0x0100 | 符号を左揃え、数値を右揃えで出力 |
  | ios::dec | 0x0200 | 10進数で出力およびデフォルトに設定 |
  | ios::oct | 0x0400 | 8進数で出力 |
  | ios::hex | 0x0800 | 16進数で出力 |
  | ios::scientific | 0x1000 | 実数に対して科学表記法に従って出力 |
  | ios:fixed | 0x2000 | 実数を小数形式で出力 |
  | ios::boolalpha  | 0x4000 | 設定された場合、trueを「true」、falseを「false」として出力<br/> 出力されない場合、整数の1と0として出力 |

  <br/>

  - フラグを設定するメンバー関数

    - long setf(long flags)
 - flagsをストリームのフォーマットフラグとして設定し、以前のフラグを返す
 
 - long unsetf(long flags)
 - flagsに設定されたビット値をストリームのフォーマットフラグから解除し、以前のフラグを返す
 
 **例**
    
    ```c++
    int main()
    {
        cout << 30 << endl; //10진수 출력
        cout.unsetf(ios::dec); //10진수 해제
        cout.setf(ios::hex); //16진수로 설정
        cout << 30 << endl;
    
        cout.setf(ios::showbase); //16진수로 설정
        cout << 30 << endl;
    
        cout.setf(ios::uppercase); //16진수의 A~F는 대문자로 출력
        cout << 30 << endl;
    
        cout.setf(ios::dec | ios::showpoint); //10진수 표현과 동시에 소수점 이하 나머지는 0으로 출력
    
        cout << 23.5 << endl;
    
        cout.setf(ios::scientific); //실수를 과학산술용 표현으로 출력
        cout << 23.5 << endl;
    
        cout.setf(ios::showpos); //양수인 경우 + 부호도 함께 출력
        cout << 23.5;
    }
    ```
 
 **出力**
 
 ```
    30
    1e
    0x1e
    0X1E
    23.5000
    2.350000E+01
    +2.350000E+01
    ```
