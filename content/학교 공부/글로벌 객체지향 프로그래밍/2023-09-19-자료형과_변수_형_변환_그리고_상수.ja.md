## データ型

**プログラムでのデータの保存形式**

## データ型の種類

### 基本データ型

* **整数** : int

* **文字** : char
  - 「a」のように、1文字のみ保存可能

* **文字列** : string
  - charとは異なり、「abc」のように複数の文字を格納可能

* **浮動小数点数**: double, float
  - doubleは小数部をより大きく保存可能
  - floatは値を宣言する際に**f**を使用する必要がある
  - doubleは接尾辞なし、または**d**を使用
* **論理データ**: bool
  - trueまたはfalseのいずれかの値をとる

<br/>

### ユーザー定義データ型

- クラス、構造体

  <br/>



## 変数

### 概念：特定のタイプのデータを保存できる領域

- クラスローダーを通じて、クラス、静的変数、およびメソッドのメモリを割り当てる（データ領域）
- **new** ステートメントを通じて生成されたオブジェクトのメモリを割り当てる<br/>(静的変数がない場合、オブジェクトごとに生成される)

- Staticが存在しない場合、オブジェクトが管理し、オブジェクトが生成されるたびに生成される

- 「static」が付いている場合、クラスが管理し、1回だけ生成される

<br/>

### 変数名

* 英字で始まる <br/>
    * 数字や特殊文字は使用不可
* 変数名は、意味が分かりやすいように、名詞、または名詞と動詞を組み合わせた表現を使用する



### 宣言方法

**アクセス制限付き[^1][^2] データ型 変数名;**

<br/>

### 変数の初期化と宣言

- 変数宣言後の初期化

​    変数名 = データ値;

- 宣言と同時に初期化

  ​  アクセス制限付きデータ型 変数名 = データ値;

<br/>

## void

### 定義

- メソッドの戻り値型であり、特定の処理を実行するための複数の文の集合
- 戻り値を持たない関数である

<br/>

## メイン

- プログラムの開始点
- string[] args と共に使用

<br/>

## xxxxxxxxxx アクセス修飾子 class クラス名{    戻り値 メソッド名(パラメータ){ ...    } データ型 変数名;}c#

- 変数のデータ型が、格納しようとする値の型と異なる場合に使用される
- float型からint型への変換のように、より小さいデータ型に変換する際、値が切り捨てられることがある
- 使用例）**int a = (int)b;**

<br/>

## 定数

- 値を変更できないメモリ領域
- 宣言時に初期化されなければならない
- 使い方：**アクセス修飾子 const データ型 変数名 = 値;**

​  



[^1]: private、public、protected、internalがあり、省略した場合はprivateがデフォルト値となる ↩
[^2]: private：クラス内部からのみアクセス可能、<br/> public：外部のクラスからアクセス可能、<br/> protected：派生クラスからアクセス可能、<br/> internal：宣言されたアセンブリ内部からのみアクセス可能
