## C# Basics

- No header
  - No class definition or declaration
    - Only the class implementation exists, similar to Java

<br/>

### Class

- A tool that groups together real-world objects that serve as the subject of programming
- Consists of member variables and member functions
- Organized into blocks using the namespace keyword
- Classes can be imported using the using directive
- Public access modifier: Allows other classes to use this class

<br/>

### Object

- When an object is created, it is instantiated in memory exactly as defined by the class.
- It consists of member variables and functions, and is also referred to as an instance.
- Objects are created in separate memory spaces.
- Creation methods:
  - variableName(objectName);
  - variableName = new ClassName();

<br/>

### Members

#### Components

- Fields (member variables), methods (member functions)
- public access modifier: Allows access from all other classes
