## Types of Data Types

### Basic Data Types

* **Integer**: int

* **Character**: char
  - Can only store a single character, such as 'a'

* **String**: string
  - Unlike `char`, it can store multiple characters, such as 'abc'

* **Floating-point numbers**: double, float
  - double can store larger decimal values
  - float requires the prefix **f** when declaring a value
  - double uses no prefix or the prefix **d**
* **Boolean data**: bool
  - Takes one of two values: true or false

## Variables

### Concept: A space capable of storing data of a specific type

- The Class Loader allocates memory for classes, static variables, and methods (data area)
- The **new** statement allocates memory for objects <br/> (if there are no static objects, a separate instance is created for each object)

- If there is no static instance, the object manages it, and it is created each time the object is instantiated

- If the `static` keyword is used, the class manages the instance, and it is created only once.

<br/>

### Variable Name

* Must begin with a letter <br/>
    * Cannot contain numbers or special characters
* Variable names should be meaningful; use nouns or combine nouns with verb forms
