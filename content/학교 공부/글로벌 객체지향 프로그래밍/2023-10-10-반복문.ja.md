## ループ

### 特徴

- 指定された演算を繰り返し実行できる
  - コンピュータの最大の特長である、**迅速性**、**自動性**を活用できる

<br/>

### 種類

#### について

- 最もよく使われるループ

  ##### 使い方

~~~c#
for(変数の初期値; 変数の条件; 増減量){
  処理文;
}
~~~

##### 例

~~~c#
for(int i=0; i<5; i++){
  Console.WriteLine(i);
}
~~~

<br/>

#### 一方で

- 条件が「真」である間、繰り返し実行されるループ

##### 使い方

~~~c#
while(条件式) {
    処理文;
}
~~~

#### do-while

- while文と同様に、条件が「真」である間、繰り返し実行されるループ
- while文との違いは、do-whileは少なくとも1回は実行される点

<br/>

それぞれについて

- 配列やコレクションなどで、データを1つずつ取得する際に便利

~~~c#
foreach(データ型 変数 in コレクションまたは配列){
    処理文;
}
~~~

<br/>

#### 改行

- ループを**1つだけ**即座に抜け出す際に使用する

<br/>

#### 続き

- continue：以下の文をスキップし、直ちに次の反復を開始する
- **反復を終了しない**

<br/>

#### 戻る

- 呼び出し元に値と制御を返す

- ##### 方式

  - **return 値;** :
    - 値を呼び出し元に返し、return の後の文は実行しない
 
 - **return;** :
    - return の後の文は実行せず、そのまま戻る
