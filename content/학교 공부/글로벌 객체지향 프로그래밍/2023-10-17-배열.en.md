## Arrays

### Disadvantages of General Variables

- Only one value can be saved at a time

​  -> You need an array to store two or more values at once

<br/>

### Features

- Can store multiple values of the same data type
- Supports various data types, including int, char, float, and classes
- Variable name = array name

<br/>

### How to Declare

1) Declaring a value upon initialization

   ~~~c#
   data_type[] array_name = new data_type[size]{value1, ..., value_n}
   ~~~

   

2) Declaring values before execution

~~~c#
DataType[] arrayName = {value1, ..., valueN}
~~~

<br/>

### Approach

- **arrayName[index]**

- Arrays are created using the `new` operator, in the form `new ClassName()`
- Properties and methods can be accessed using the `**.**` operator

- Array indices start at 0, and the last element is 'arrayName.length - 1'

- Using the System.Index and ^ [^1] operators makes it easier to access arrays

  <br/>

### Parameters

#### Method

##### Definition

- Specify an array type as a parameter

  ~~~c#
  access_restrictor return_type function_name(array_type[] array_name) {} // Specify the array type as a parameter
  ~~~

<br/>

##### Call

- **MethodName(ArrayName);**
- Example

~~~c#
public static void Main(string[] args)
    {
 void Func(int[] arr)
    {
 foreach(int i in arr)
 Console.WriteLine(i);
    }
 
 int[] testArray = {1,2,3};
 Func(testArray);
    }
~~~

**Results**

~~~
1
2
3
~~~

<br/>

##### Return Type

###### Declaration

- Specify an array as the return type

~~~c#
access-modifier array-type[] function-name(parameter-type parameter-name) {}
~~~

###### Return

- Declare the array so that its type matches the type of the returned array

  ~~~c#
  arrayType[] arrayName = functionName();
  ~~~

#### Call-by-value

##### Call by Reference

- Ensuring that changes made to array values in the called method are reflected in the calling method as well

- Pass a reference to an array using the **ref** keyword

  - Share the address under a different name

- Sample code

  ~~~c#
  using System;
 
 class Program
  {
 static void ArrayIdxChange(ref char[] array, int originalIdx, int changeIdx)
 {
 char tempString = array[originalIdx];
 array[originalIdx] = array[changeIdx];
 array[changeIdx] = tempString;
 }
  
      static void Main(string[] args)
 {


char[] testArr = { 'a', 'b', 'c', 'd', 'e' };
 int originalIdx, changeIdx;
 Console.Write("What is the original position of the character to be swapped? >>");
          originalIdx = Convert.ToInt32(Console.ReadLine());
 Console.Write("What is the position of the character to be replaced? >>");
 changeIdx = Convert.ToInt32(Console.ReadLine());
 
 Console.Write("Original array >>");
          foreach (char c in testArr)
 Console.Write(c);
 Console.WriteLine();
 ArrayIdxChange(ref testArr, originalIdx, changeIdx);
 Console.Write("Modified array >>");
 foreach (char c in testArr)
 Console.Write(c);
 }
  }
  ~~~
  
  **Result**
 
 ~~~
  What is the original position of the character to be replaced? >>1
  What are the positions of the selected character and the replacement character? >>2
  Original array>>abcde
  Modified array>>acbde
  ~~~

#### Related Errors

##### 1. IndexOutOfRangeException

- Occurs when attempting to access a value at an invalid index in an array

- Example

  ~~~c#
  using System;
 
 class Program
  {
 static void ArrayIdxChange(ref char[] array, int originalIdx, int changeIdx)
 {
 char tempString = array[originalIdx];
 array[originalIdx] = array[changeIdx];
          array[changeIdx] = tempString;
 }
 
 static void Main(string[] args)
 {
 char[] testArr = { 'a', 'b', 'c', 'd', 'e' };
 ArrayIdxChange(ref testArr, 3, 6);
          Console.Write("변경된 배열>>");
 foreach (char c in testArr)
 Console.Write(c);
 }
  }
  ~~~

  **Error Results**

  ~~~
  System.IndexOutOfRangeException: 'Index was outside the bounds of the array.'
  ~~~

  <br/>

  ##### 2. NullReferenceException

  - Occurs when attempting to access a method or property without specifying an array or an array element
  - Sample code

  ~~~c#
  using System;
 
 class Program
  {
 static void ArrayIdxChange(ref char[] array, int originalIdx, int changeIdx)
 {
 char tempString = array[originalIdx];
 array[originalIdx] = array[changeIdx];
 array[changeIdx] = tempString;
 }
  
      static void Main(string[] args)
 {
 char[] testArr = null;
 ArrayIdxChange(ref testArr, 3, 6);
 Console.Write("변경된 배열>>");
 foreach (char c in testArr)
 Console.Write(c);
 }
  }
  ~~~

  **Error Results**

  ~~~
  System.NullReferenceException: 'Object reference not set to an instance of an object.'
  ~~~

  <br/>

  ##### 3. ArgumentException

  - Occurs when initializing an array or passing incorrect arguments to a method

  <br/>

  ##### 4. ArrayTypeMismatchException

  - Occurs when attempting to change the array's type in an invalid manner

### Array Class

#### Definition

- Provides methods for creating, manipulating, searching, and sorting arrays, enabling it to serve as the base class for all arrays in the Common Language Runtime[^2]

  <br/>

#### Properties

| Name | Meaning |
| :------------- | :----------------------------------------------------------- |
| IsFixedSize    | Returns a value indicating whether the array size is fixed |
| IsReadyOnly    | Returns a value indicating whether the array is read-only |
| IsSynchronized | Returns a value indicating whether access to the array is synchronized and thread-safe |
| Length | Returns the size of the array |
| LongLength     | Returns a 64-bit integer representing the total number of elements across all dimensions of the array |
| MaxLength | Retrieves the maximum number of elements the array can contain |
| Rank | Returns the number of dimensions of the array <br> Ex) 1-dimensional: 1<br/>2-dimensional: 2     |
| SyncRoot | Retrieves the object used to synchronize access to the array |

<br/>

| Method Name | Description |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| Clear(Array) | Deletes each element of the array |
| Clear(Array, Int32, Int32) | Sets the array size to the default value of the type of each element     |
| Clone() | Creates a partial copy of the array |
| ConstrainedCopy(Array, Int32, Array, Int32, Int32) | Copies a range of the array starting from the specified index and pastes it into another array starting from the specified destination index |
| ConvertAll<TInput, TOutput>(TInput[], Converter<TInput, TOutput>) | Converts the selected array to an array of a different type |
| Copy(Array, Array, Int32) | Copies the array starting from the first element and pastes it into another array starting from the first value<br/>The length is specified as a 32-bit integer |



[^1]: You can index the collection starting from the last element in reverse order
