## The Program Development Process

### Developers vs. Computers

- Developer: Can understand numbers, letters, and special characters
  - Computer: Can only read binary numbers

<br/>

### Programming Languages

#### Concept

- A language that allows humans to give commands to a computer using English words that people can understand
- It has a set of rules

<br/>

### Machine Language

- This is the language used by computers; the CPU interprets this machine code to perform tasks

<br/>

### Build

- The process of generating an executable file
- When the build is complete, a file newly translated into machine code is generated
  - Also known as an object file
- Linking is the process of combining multiple object files into a single program
- In C#, programs provide commonly used functions in advance; these are called libraries
- To create an executable file, you must link the object files that provide these functions

<br/>

### Compilation Errors and Linking Errors

#### Compilation Error

- This is an error that occurs during the process of converting source code into machine code; it is a syntax error.

#### Link Error

- Errors that occur when linking source files or source libraries
- It is necessary to verify that the source definitions are correct and that the calls are made correctly

#### Logical errors after the executable file is generated

- If the desired results are not achieved
  - The issue can be resolved through debugging 

<br/>

### Reserved Words

- Keywords that are predefined in a programming language for specific purposes and cannot be used for any other purpose
- Developers can define namespace names, class names, method names, and variable names

<br/>

### Namespace

#### Definition

- When working collaboratively, there may be conflicts over class names, method names, variable names, and so on.
- To prevent such conflicts, group your code together.

#### How to Use

~~~c#
namespace namespaceName {
class className {
  ...
    }
}
~~~

<br/>

- When using this class from another class, you can do so by using **using namespace**

<br/>

### Class

#### Definition

- A collection of objects that are the subject of programming, organized using the keyword "class"
- A class contains methods that represent behavior, as well as fields that hold attributes and member variables

#### How to Use

~~~c#
access-modifier class ClassName {
  return-type MethodName(parameters) {
    ...
  }
 
 data-type variable;
}
~~~
