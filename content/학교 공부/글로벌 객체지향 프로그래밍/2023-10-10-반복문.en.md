## Loops

### Features

- Capable of repeatedly performing given operations
  - Capable of leveraging **speed** and **automation**, the most powerful features of computers

<br/>

### Types

#### for

- The most commonly used loop

  ##### How to Use

~~~c#
for(initial value; condition; increment){
  statement;
}
~~~

##### Example

~~~c#
for(int i=0; i<5; i++){
  Console.WriteLine(i);
}
~~~

<br/>

#### while

- A loop that repeats as long as the condition is true

##### How to Use

~~~c#
while (condition) {
    statement;
}
~~~

#### do-while

- A loop that repeats as long as the condition is true, similar to a while loop
- The difference from a while loop is that a do-while loop executes at least once

<br/>

foreach

- Useful for retrieving data one item at a time from arrays, collections, and similar structures

~~~c#
foreach (data_type variable in collection or array) {
    statement;
}
~~~

<br/>

#### break

- Use this to **immediately** exit a single loop

<br/>

#### continue

- continue: Skips the following statement and proceeds directly to the next iteration
- **Does not terminate the loop**

<br/>

#### return

- Returns the value and control to the caller

- ##### Method

  - **return value;** :
    - Returns the value to the caller and does not execute the statements following `return`
 
 - **return;** :
    - Returns the value without executing the statements following `return`
