## 条件文

### 種類

#### もし

- 条件が満たされた場合、角括弧内のコードを実行する

- 使用方法：

  ~~~c#
  if(条件){
 
 }
  ~~~

- {}は、if文内で実行する命令が1行のみの場合、省略可能である

<br/>

#### もしそうでないなら

- 条件を満たす場合、If文内のコードを実行する

- 条件を満たさない場合、else文内の文を実行する

- 使い方：

  ~~~c#
  if(条件){
 
 }
  else{
 
 }
  ~~~

<br/>

#### if、else、else if

- 条件を満たした場合、if文の括弧内のコードを実行する
- 条件を満たさない場合、else if文を実行し、その条件を満たす場合は、else if文の括弧内のコードを実行する
- if文とelse if文のいずれも条件を満たさない場合、else文を実行する
- 使用方法

~~~c#
if(条件1)
{

}
else if(条件2)
{

}
else{
 
}
~~~

<br/>

#### switch-case

- Switch文に変数を挿入し、caseに条件を設定することで、各条件が満たされた場合に、そのcase内の文を実行する
- if文と同様に、実行する文が1行の場合、{}を省略できる
- 文が複雑になるほど、if文よりもパフォーマンスが向上する
- 使用方法

~~~c#
switch(変数名){
    case 条件: {} break;
 ...
    case default: {} break; //省略可能
}
~~~

<br/>

### 演算子

#### 一般的な演算子

##### ==

- 左辺の変数と右辺の変数の値が同じかどうかを返す

- 例：

  ~~~c#
  if(a == b) Yes();
  else No();
  ~~~

<br/>

##### !=

- 左辺の変数と右辺の変数の値が異なるかどうかを返す

- 例：

  ~~~c#
  if(a == b) Yes();
  else No();
  ~~~

<br/>

##### <

- 左辺の変数の値が右辺の変数の値より小さいかどうかを返す

- 例：

  ~~~c#
  if(a < b) Yes();
  else No();
  ~~~

<br/>

##### <=

- 左辺の変数の値が右辺の変数の値以下であるかどうかを返す

- 例：

  ~~~c#
  if(a <= b) Yes();
  else No();
  ~~~

  

<br/>

##### >

- 左辺の変数の値が右辺の変数の値よりも大きいかどうかを返す

- 例：

  ~~~c#
  if(a > b) Yes();
  else No();
  ~~~

  

<br/>

##### >=

- 左辺の変数の値が右辺の変数の値以上であるかどうかを返す

- 例：

  ~~~C#
  if(a >= b) Yes();
  else No();
  ~~~

<br/>

#### 多重比較演算子

##### &&

- …の役割を担う

- 例：

  ~~~c#
  if(a == 1 && b == 3) Console.WriteLine("True");
  ~~~

  **結果**

  ~~~
  True
  ~~~

  

  <br/>

##### ||

- またはその役割を担う

- 例：

  ~~~c#
  int a = 1;
  int b = 5;
  if(a == 1 || b == 3) Console.WriteLine("True");
  ~~~

  **結果**

  ~~~
  True
  ~~~

<br/>

## クラス

### 定義

プログラムの対象となる事物を、「class」というキーワードを用いて、メンバ関数とメンバ変数で実装したもの

### オブジェクト

#### 定義

- 作成されたクラスを「new」キーワードを使用して、ヒープメモリ上に生成すること

#### 使い方

~~~c#
クラス名 オブジェクト名 = new クラス名();
~~~

##### 例

~~~c#
Circle circle = new Circle();
~~~

<br/>

#### 呼び出し方法

- Static 予約語がない場合

  - **オブジェクト名.メンバー;**
  - 例）

  ~~~c#
  dog.name;
  ~~~

- Static 予約語がある場合

  - **クラス名.メンバー;**
  - 例）

  ~~~c#
  Dog.AnimalName
  ~~~


<br/>

### これ

- 自身のオブジェクトのメンバにアクセスする際に使用する
- オブジェクトの外部からオブジェクトのメソッドや変数にアクセスする際は、**オブジェクト名.メソッド** または **オブジェクト名.変数** を使用してアクセスする
- オブジェクトの内部から自身の変数やメソッドにアクセスする際に使用する

<br/>

### コンストラクタ

- 初期化のためのメソッドであり、インスタンス生成時に呼び出される
- クラス名と同じ名前を使用し、引数として任意の値を受け取ることができる
- 戻り値がない

<br/>

名前空間

#### 定義

- 関連するクラスのグループ

#### 使用目的

- 共同作業を行う際、自分のクラスを区別するために使用する

#### アプローチ

- **名前空間.クラス.メソッド/メンバー変数**
