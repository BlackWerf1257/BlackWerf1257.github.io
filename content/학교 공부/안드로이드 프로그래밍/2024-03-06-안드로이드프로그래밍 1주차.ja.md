### Android Studioでボタンを作成する

- コードは次のように構成した

~~~xml
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:context=".MainActivity">


    <Button
 android:id="@+id/btn"
 android:layout_width="match_parent"
 android:layout_height="wrap_content"
 android:text="ボタン"
 android:textSize="30sp" />
    <Button
 android:id="@+id/btn2"
 android:layout_width="match_parent"
 android:layout_height="wrap_content"
 android:text="ボタン2"
 android:textSize="30sp" />

</LinearLayout>
~~~

**ボタン作成に使用された要素**

- **android:id**
  - ボタンを作成した後、外部クラスからこのボタンにイベントを適用するために使用する
- LinearLayoutは、各ビューを1つの列または行に並べて表示するレイアウトである
- **android:layout_width**
  - match_parent、wrap_parentのいずれかを選択可能
  - match_parent：ボタンの幅がビューの幅に合わせて拡張される
  - wrap_parent : ボタンの幅が、ボタン内の文字のサイズ分だけ拡張される
- **android:layout_height**
  - match_parent、wrap_parent のいずれかを選択可能
  - match_parent : ボタンの高さが親（ビュー）の高さ分だけ拡張される
    - ただし、途中に他のオブジェクトが存在する場合、そのオブジェクトの手前までしか高さが伸びない。
  - wrap_parent : ボタンの高さが、ボタン内の文字が占める高さ分だけ拡張される
- **android:text**
  - 出力するテキストを入力するために使用される
- **android:textSize**
  - ボタンのサイズを指定するために使用される
  - 数値 + sp の形式で指定する

### ボタンへのイベント実装

- ボタンにイベントを実装するには、大きく分けて3つの手順がある
  1) XMLファイルでボタンを作成し、IDを指定する
 - この際、IDの形式は**@+id/ID名**とする
  2) Javaファイルでボタンを宣言し、メモリに割り当てる
 - ボタンの宣言は**Button ボタン名;**という形式で行う
     - ボタンをメモリ上に宣言するには、onCreate関数内で**buttonName = (Button)findViewById(R.id.*buttonId*);**という形式を使用する
  3) onCreate関数内でbuttonName.setOnClickListener(new View.OnClickListener())を使用して、実行するイベントを指定する

<br/>

サンプルコードは以下の通りです

~~~xml
<!-- activity_main.xml -->
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <Button
 android:id="@+id/testButton"
 android:layout_width="match_parent"
 android:layout_height="wrap_content"
 android:text="Test Button"/>

</LinearLayout>
~~~

~~~java
//MainActivity.java

package kr.ac.baekseok.testpr;

import androidx.appcompat.app.AppCompatActivity;

import android.os.Bundle;
import android.view.View;
import android.widget.Button;
import android.widget.Toast;

public class MainActivity extends AppCompatActivity {
    Button testBtn;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
 super.onCreate(savedInstanceState);
 setContentView(R.layout.activity_main);
 testBtn = (Button)findViewById(R.id.testButton);

        testBtn.setOnClickListener(new View.OnClickListener() {
 @Override
 public void onClick(View v) {
 Toast.makeText(getApplicationContext(), "クリックされました", Toast.LENGTH_LONG).show();
 }
 });
    }
}
~~~
