### Creating a Button in Android Studio

- The code was structured as follows

~~~xml
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:context=".MainActivity">


    <Button
 android:id="@+id/btn"
 android:layout_width="match_parent"
 android:layout_height="wrap_content"
 android:text="버튼"
 android:textSize="30sp" />
    <Button
 android:id="@+id/btn2"
 android:layout_width="match_parent"
 android:layout_height="wrap_content"
 android:text="버튼2"
 android:textSize="30sp" />

</LinearLayout>
~~~

**Elements used to create the button**

- **android:id**
  - Used after creating a button to apply events to it from an external class
- LinearLayout is a layout that arranges individual views in a single column or row
- **android:layout_width**
  - You can choose between `match_parent` and `wrap_parent`
  - `match_parent`: The button's width expands to match the width of the containing view
  - wrap_parent: The button’s width expands to fit the size of the text inside it
- **android:layout_height**
  - You can choose either `match_parent` or `wrap_parent`
  - match_parent: The button’s height expands to match the height of its parent (view)
    - However, if another object is positioned in the middle, the height only increases up to that object.
  - wrap_parent: The button's height expands to match the height of the text inside the button.
- **android:text**
  - Used to specify the text to be displayed.
- **android:textSize**
  - Used to specify the size of the button.
  - Specified in the format of a number followed by "sp".

### Implementing Events on Buttons

- There are three main steps to implementing an event for a button
  1) Create the button in the XML file and assign an ID to it
 - The ID format must be **@+id/id_name**
  2) Declare the button in the Java file and assign it to memory
 - Declare the button using the format **Button buttonName;**
     - To declare the button in memory, use the following syntax in the onCreate method: **buttonName = (Button)findViewById(R.id.*buttonId*);**
  3) Within the onCreate method, specify the event to be executed using buttonName.setOnClickListener(new View.OnClickListener());

<br/>

The sample code is as follows

~~~xml
<!-- activity_main.xml -->
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <Button
 android:id="@+id/testButton"
 android:layout_width="match_parent"
 android:layout_height="wrap_content"
 android:text="Test Button"/>

</LinearLayout>
~~~

~~~java
//MainActivity.java

package kr.ac.baekseok.testpr;

import androidx.appcompat.app.AppCompatActivity;

import android.os.Bundle;
import android.view.View;
import android.widget.Button;
import android.widget.Toast;

public class MainActivity extends AppCompatActivity {
    Button testBtn;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
 super.onCreate(savedInstanceState);
 setContentView(R.layout.activity_main);
 testBtn = (Button)findViewById(R.id.testButton);

        testBtn.setOnClickListener(new View.OnClickListener() {
 @Override
 public void onClick(View v) {
 Toast.makeText(getApplicationContext(),"Clicked", Toast.LENGTH_LONG).show();
 }
 });
    }
}
~~~
