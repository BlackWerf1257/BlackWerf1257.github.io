## Class Topic

**Sensor**

### Sensor Hardware

- Smartphones are equipped with sensors such as gyroscopes, proximity sensors, accelerometers, and compasses, which are useful for monitoring three-dimensional motion and the surrounding environment.

#### Types of Sensors

##### Motion Sensor

- Measures acceleration and rotational force along three axes
- Types: **accelerometer, gravity sensor, gyroscope, rotation vector sensor**

##### Environmental Sensors

- Measures various environmental factors such as ambient temperature, pressure, illuminance, and humidity
- Types: **barometer, lux meter, thermometer**

##### Position Sensor

- Measures the physical location of the device
- Types: **Orientation sensors, magnetometers**

<br>

### Android Sensor Classes

#### SensorManager Class

##### Features

- Provides a list of sensors built into the device
- Provides sensor-related methods, such as accessing sensors and registering events

#### Sensor Class

##### Features

- Creates an instance of a specific sensor

#### SensorEvent Class

##### Features

- Creates a sensor event object that provides information about the sensor event

#### SensorEventListener Class

#####  Features

- Using the interface, you can create methods that are called when the sensor's value or accuracy is changed.
