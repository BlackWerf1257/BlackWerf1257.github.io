## Sorting

### Basic Sorting Algorithms

- On average, it takes $$ θ(n^2) $$ units of time

#### Types

- Selection sort
- Bubble sort
- Insertion sort

#### Selection Sort

- Search for the maximum value in each loop
- Swap the maximum element with the **rightmost** element, then remove the rightmost element
- Repeat the above loop until only one element remains
- Running time: $$ θ(n^2)$$

#### Bubble Sort

- Compare the sizes of the two adjacent values
- If the value on the left is larger, swap it with the value on the right
- Time complexity: $$ θ(n^2)$$
- The value on the far right is excluded from the sorting process

#### Insertion Sort

- Unlike selection or bubble sort, this involves **increasing** the size of an **already sorted array** by one element at a time
- Insert the value at the corresponding position
- If the array is sorted up to A[1, n], inserting a row results in the array being sorted up to A[1, ..., n+1]
- Time complexity:
  Worst Case / Average Case: $$θ(n^2)$$<br>Best Case: $$θ(n)$$

<br>

### Advanced Sorting Algorithms

#### Types

- On average, it takes $$θ(nlog^n)$$ units of time

- Merge sort
- Quick sort
- Heap sort

#### Merge Sort

- Split the array in half
- Sort each half using a recursive function, then merge them
- Time complexity: $$θ(nlog^n)$$
