## Greedy Algorithm

- Algorithms that select only the immediate benefit
- Most are far from the optimal solution
  - **However**, there are rare cases where the optimal solution is guaranteed
- **Minimum spanning trees** are a type of greedy algorithm

### Cases Where an Optimal Solution Is Not Guaranteed

1. Finding the optimal path in a binary tree
2. The knapsack problem
3. The coin-flipping problem

### Cases Where an Optimal Solution Is Guaranteed

1. Prim’s Algorithm and Kruskal’s Algorithm
2. Conference Room Allocation Problem
   - What is the solution when there is only one conference room and multiple requests have been made?
 - Among the requests that specify start and end times, **allocate the rooms in order of the earliest end time**
