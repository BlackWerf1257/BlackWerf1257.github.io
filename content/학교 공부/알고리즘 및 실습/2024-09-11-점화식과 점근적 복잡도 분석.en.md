## Analysis of Ignition Functions and Asymptotic Complexity

### Lighting Ceremony

#### Definition

- A function expressed in terms of a function of a smaller variable (a recursive function)
- Recurrence relation: $$T(n) = 2T(n/2) + overhead$$[^1]

- The runtime of a merge sort on a set of size n is the sum of the time required to perform two merge sorts on sets of size n/2, plus any additional overhead.

#### Asymptotic Analysis Methods for Ignition Processes

##### Iterative Replacement

- A method of repeatedly replacing the function with smaller problems

###### Implementation Process

###### Binary Search

- To search for data stored in an array, the data is located by repeatedly comparing it with the middle position of the array.
- After each comparison, the next data item to be compared is the one at the 1/2 position of the current data.
  - If there are n items, the positions are n/2, n/4, n/8, and so on.
  - If there are 1,000 items, the data can be found within approximately 10 operations: $$log2^{1000}$$
