### Asymptotic Complexity and Degree of Algorithms

#### Sequential vs. Constant

- When c1 and c2 are constants, comparing c1^n and c2^n shows that the **degree** is important.
- The constant terms can be ignored; in summary, the expression is evaluated as N approaches infinity. 

### Asymptotic Notation

1. **$$O(f(n))$$**

- A function that grows at a rate of **at most** f(n)

- Asymptotic upper bound

- Example

  $$ O(n),⠀⠀O(nlog n),⠀⠀O(n^2) $$⠀

- It is customary to write $$g(n) \in O(f(n))$$ as $$g(n) = O(f(n))$$

- Important Notes

  - When writing, keep it as concise as possible

    - Example)

      ##nlogn + 5n = O(nlogn)$$, but there is no need to write it as $$O(n^2)$$
  
- Example

  - Quick sort ($$O(n^2)$$)
  - Heap sort ($$O(nlogn)$$)
  

2. **$$Ω(f(n))$$**

   - Definition

     - **At least**, a function that grows at a rate of f(n)

     - Symmetric with respect to $$O(f(n))

     - Asymptotic lower bound

   - Example
 $$5n^2 + 4n = \Omega(n^2)$$

     $$g(n) = \Omega(f(n))$$

     - g increases faster than f

     - For an input size of n, the runtime of g(n) is greater than that of f(n)

3. **$$Θ(f(n))$$**

   - Definition

     - A function that grows like f(n)

     - In $$g(n) = Θ(f(n))$$, the growth rate of g is equal to the growth rate of f

     - Asymptotic equivalence
 
 - Examples
 - Selection sort ($$Θ(n^2)$$)
 - Average ($$Θ(nlogn)$$)


### Binary Search

#### Definition

- An algorithm that searches by comparing values and halving the search range until the target value is found

#### Analysis Method

- Searches for a specific value in a **sorted** list and returns its position; if the value is not found, returns -1

- Binary search refers to the process of dividing data into two parts to find a value

#### Features

- More efficient than linear search
  - Assuming there are 1,000 items, linear search examines every value, so it runs at most 1,000 times, whereas binary search runs at most 10 times <br>$$log2^1000 ≒ 10$$
- The complexity of binary search is $$O(log n)$$, while that of linear search is $$O(n)$$

#### Examples
- Looking up a word in a dictionary
- Finding a room number at a hotel
