## Mutually Exclusive Sets

### Types

Processed using a linked list or tree

The method using a tree is more efficient

This post omits operations on sets using linked lists.

### Methods for Processing Sets Using Trees

- Elements belonging to the same set are managed as a single tree
  - Child nodes point to their parent nodes
- The root of the tree is designated as the **representative element** of the set

### Types

- **Make-set**
  - An operation that creates a set consisting of x elements
- **Find-set**
  - An operation used to determine the set to which x belongs

- **Union**
  - The union operation that combines sets x and y
