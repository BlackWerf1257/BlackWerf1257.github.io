## List

### Circular Linked List

#### Definition

- The value of the last node's link is not null; instead, it holds the address of the first node
- A linked list in which the last node points to the first node

## Graph

#### How to Use the Graph

##### All Bridges

### Definition

$$ G = (V, E) $$

- V(G): A set of one or more vertices
- E(G): A set of edges consisting of pairs of vertices
- Both sets are finite

<br>

### Categories

#### Undirected Graph

- There is no order on the pairs of vertices representing edges

  $$ (V_1, V_2) = (V_2, V_1) $$

#### Directed Graph

- Arrows indicate the direction of the lines

- In a directed graph, the order of the vertices in a pair is important

- Graph representation: $$ <V_1, V_2> $$

  $$ V_1: Tail of the edge \; V_2: Head of the edge $$

  $$ <V_1, V_2> \; \ne \; <V_2, V_1> $$

#### Mixed Graphs (Undirected + Directed)

### Types

#### Weighted Graph

- A graph with weighted edges

####  Multi Graph

- **Contains at least one edge** between two vertices
- Properties of directed graphs
  - The weight assigned to an edge

#### Single Graph

- Contains either 0 or 1 **edge** between any two vertices

#### N-graph

- Contains only isolated vertices
- The set of edges is empty
- Isolated vertex
  - A vertex that is not connected to any other vertex

#### Colored Graph

- A graph that uses lines of different colors or shapes to distinguish between relationships

#### Complete Graph

- The graph with the maximum number of edges in an undirected graph with n vertices

<br>

### Related Terms

#### Incident

- Given an edge $$ e \in E $$ connecting the pair of vertices $$ (V_1, V_2) $$ , the vertices $$ V_1 $$ and $$ V_2 $$ are connected by the edge $$ e $$

#### Adjacent

- Two vertices connected by an edge are adjacent

#### Path

- A path from one vertex to another
- A sequence of edges in which the starting point of each edge is the ending point of the next edge

#### Length

- Number of edges along the path

#### Simple Path

- All vertices on a single path are distinct

#### Elementary Path

- All edges on a single path are distinct

#### Cycle

- A simple path with the same starting point and ending point

#### Cyclic Graph

- A graph containing a cycle

#### Indegree

- The number of edges **leading to** a given vertex

#### Outdegree

- The number of edges **originating from** a given vertex

#### Total Degree

- The **sum of the in-degree and out-degree** at a vertex

#### Walk

- A vertex that is both the start and end point of an edge
- A path of length 1

#### Subgraph

- A graph consisting of a subset of the vertices and edges of graph G

#### Acyclic Graph

- A graph with no cycles

#### Dag(Directed Acycic Graph)

- Directed acyclic graph

#### Connected

- In an undirected graph G, if there is a path from vertex $$ V_p \rightarrow V_q$$, then $$V_p$$ and $$V_q$$ are connected.

#### Connected Graph

- The graph when any two vertices $$ V_p, V_q$$ are connected

### Methods of Expression

#### Adjacency Matrix

##### Assumptions

- The graph G = (V, E) contains n vertices

##### Adjacency Matrix

- A 2-dimensional array of nxn
- An element a of the array

$$ a_{ij} = \begin{cases} 1 \qquad a_{ij} \in E \\ 0 \qquad ((V_i, V_j) \notin E) \end{cases}$$

- A square matrix with an equal number of rows and columns
- All elements are either 0 or 1
- Using the adjacency matrix, it is easy to determine whether there is an edge connecting any two vertices i and j
- The number of elements with a value of 1 in the i-th row = the out-degree of vertex $$V_i$$

<br>

#### Adjacency List

- Represent the n rows of the adjacency matrix as n linked lists

- There is one linked list associated with each vertex of the graph

- Each node in list i

  - Vertices adjacent to vertex i

- Each node contains two fields

  1) Vertex field: The index of the vertex
  2) Link field: The next adjacent vertex

  <br>

- Out-degree of a vertex
  - Number of nodes in the corresponding adjacency list
- In-degree of a vertex
  - Using the reverse adjacency list or the orthogonal list

##### List of Stations Nearby

- Consists of nodes representing all adjacent nodes entering a specific node
- Constructs an adjacency list for the edges entering a node to calculate the in-degree
- A linked list for each column in the adjacency matrix

<br>

#### Adjacent Multiple Lists

- Degree of multiplicity or weight: $$W_{ij}$$
- The value of \(a\) for each element:

$$ a_{ij} = \begin{cases} W_{ij} \qquad ((V_i, V_j) \in E) \\ 0 \qquad ((V_i, V_j) \notin E) \end{cases}$$
