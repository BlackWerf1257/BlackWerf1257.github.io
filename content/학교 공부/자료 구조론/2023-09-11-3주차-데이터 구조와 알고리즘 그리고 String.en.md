#### File Structure

##### Live

- A method of storing data on a storage device in physical order

##### Sequential

- A method in which the key value of a record is used as input for a hash function, and the file is stored such that the hash result becomes the address of the record with that key value
- **Hashing required**
- HF (record key value) -> Address

##### Indexed Sequential

- The file consists of an index and a sequential data area
- Data can be accessed in two ways
    1) Sequential access
    1) Random access via the index
- When sorting, binary search is possible, which improves efficiency
  - Binary search: Dividing the data into two search areas

### String

#### Definition

- A sequence of characters

#### Purpose

- Basic elements of a program
  - Label, Identifier, Reserved Word, User-Defined Word....
- Basic elements of a document

#### Types

##### Sequential String (Uncompressed)

###### Definition

- A method of representing consecutive characters in consecutive words
- Each word represents a single character

###### Advantages

- Fast write speeds

###### Disadvantages

- Since it is an uncompressed format, memory efficiency is reduced

###### Example

- S = 'DATA'

- Expression method:

  ~~~
  oooDoooAoooToooA
  ~~~

##### Fixed-length sequential string

###### Compressed String

- A method of representing a string as a sequence of words
- Each word represents two or more characters
  - Enables efficient memory usage

- Example

  - S="DATE"

    Data / | | | | | | | | | | |

###### Related Links

- Uses linked blocks

  - Each block contains characters

- A single block can be divided into two areas

  - **Data area**: Contains one or more characters
  - **Link area**: Contains the address of a block containing the following characters

- Characters are stored in blocks, which are non-contiguous memory locations

- **Memory usage decreases** due to the use of pointers

- Very useful for **operations that change the length of a string**

  - Examples: Insertion, deletion, substring

- Example) S="DATASTRUCT"

##### Variable-length String

- The string length is variable

###### Related Operations

- Concatenation
  - Creates a single composite string by joining two strings together
- Insertion
  - Adds a new string between two characters in a string
- Deletion
  - Removes one or more characters from a string
- Replace
  - Replaces a character in a string with another character
- Substring
  - Finds a portion of consecutive characters within a string to create a new string
- Indexing
  - Returns the position of a character or substring within a string as an integer
  - Returns **0** if the character does not exist
  - If multiple occurrences exist, returns the **first value**
- Pattern matching operation
  - Searches for a specified substring within a string
    - Returns **True** if the value exists, and **False** if it does not

<br>

- They all share the common feature of being consecutive words
- They are used based on considerations such as processing time, memory efficiency, and ease of use
