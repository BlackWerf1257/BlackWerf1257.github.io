## Collision

### Configuring Collisions

#### How to Set Up

- Go to **Project Settings** -> **Collision** tab

#### Object Channel

##### Types

- Ignore
  - Ignore collisions
- Overlap
  - Overlap upon collision
- Block
  - Objects bounce off each other upon collision

#### Presets

- No Collision
- Query Only (No Physics Collision)
  - Unity's OnTriggerEnter
- Physics Only (No Query Collision)
  - Handles physics only
- Collision Enabled (Query and Physics)
  - Handles both queries and physics
