#### std

- C++의 표준 namespace
- iostream 헤더파일 내부에 선언됨
- 관련 함수로는 **cout, cin, end** 등이 존재함
- Using을 이용하여 std를 선언할 경우 생략가능



### 컴퓨터의 최소 정보

- 비트 : 컴퓨터에서 정보 처리를 위한 최소 단위
- 최적화된 비트 : CPU에서 한번에 처리 가능한 단위



### 변수 선언

- C++의 변수 선언 :

​	데이터를 유지하기 위한 공간

​	위치 제한이 X, 함수의 상위 위치 권장



### 명령어

- Strcmp(): 문자열 비교
- Strlen(): 문자열 길이
- Strcyp(): 문자열 복사



### String 클래스

- Char와 다르게 문자열 크기 제한이 없음
