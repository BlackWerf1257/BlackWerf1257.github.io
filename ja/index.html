Clean Theme, Tranquil Feel

ようこそGoyoへ！

韓国の言葉「ゴヨ」（고요）——静けさや穏やかさを意味する——に着想を得たGoyoは、シンプルさと洗練されたドキュメント作成を目指すZolaテーマです。Goyoを使えば、美しく実用的なドキュメントページを簡単に作成できます。

清潔で、ミニマルで、コンテンツに焦点を当てた

ドキュメント対応

Provides a clean writing experience for documentation.

シンプルなデザイン

A theme that pursues minimalism.

カスタマイズと洗練されたランディングページ

Highly customizable with a beautiful landing page.

SEO最適化済み

Provides a structure optimized for search engines.

各種ショートコード

Offers a variety of useful shortcodes.

ダークモードとライトモード

Supports both dark and light modes.

技術スタック

簡単な設置

お好みの方法で、わずか数秒でGoyoを始めましょう

git clone https://github.com/hahwul/goyo.git themes/goyo
git submodule add https://github.com/hahwul/goyo.git themes/goyo
curl -sL https://github.com/hahwul/goyo/archive/refs/heads/main.zip -o goyo.zip
